เมทริกซ์

เรียนรู้เมทริกซ์ การดำเนินการ ดีเทอร์มิแนนต์ เมทริกซ์ผกผัน และการประยุกต์ใช้ในการแก้ระบบสมการ

1. ความรู้เบื้องต้นเกี่ยวกับเมทริกซ์

เมทริกซ์คือการจัดเรียงจำนวน สัญลักษณ์ หรือการแสดงออก ในรูปแบบสี่เหลี่ยมผืนผ้าที่มีแถวและหลัก องค์ประกอบแต่ละตัวในเมทริกซ์เรียกว่าสมาชิกหรือตัวประกอบ

นิยามเมทริกซ์

เมทริกซ์ A ขนาด m × n

A = [aᵢⱼ] =
⎡ a₁₁ a₁₂ ... a₁ₙ ⎤
⎢ a₂₁ a₂₂ ... a₂ₙ ⎥
⎢ ⋮ ⋮ ⋱ ⋮ ⎥
⎣ aₘ₁ aₘ₂ ... aₘₙ ⎦

โดย m = จำนวนแถว, n = จำนวนหลัก

ประเภทของเมทริกซ์

เมทริกซ์สี่เหลี่ยมจัตุรัส

m = n (จำนวนแถวเท่ากับจำนวนหลัก)

เมทริกซ์แถว

m = 1 (มีแถวเดียว)

เมทริกซ์หลัก

n = 1 (มีหลักเดียว)

เมทริกซ์ศูนย์

สมาชิกทุกตัวเป็น 0

เมทริกซ์หนึ่ง (I)

เส้นทแยงมุมเป็น 1, อื่น ๆ เป็น 0

เมทริกซ์ทรานสโพส (A^T)

แถวกลายเป็นหลัก หลักกลายเป็นแถว

2. การดำเนินการของเมทริกซ์

การบวกและลบเมทริกซ์

เงื่อนไข: เมทริกซ์ต้องมีขนาดเท่ากัน

A ± B = [aᵢⱼ ± bᵢⱼ]
ตัวอย่าง: การบวกเมทริกซ์
A = ⎡ 2 3 ⎤ B = ⎡ 1 4 ⎤
⎣ 1 5 ⎦ ⎣ 2 3 ⎦

A + B = ⎡ 3 7 ⎤
⎣ 3 8 ⎦

การคูณเมทริกซ์ด้วยสเกลาร์

kA = [kaᵢⱼ]

การคูณเมทริกซ์

เงื่อนไข: จำนวนหลักของ A = จำนวนแถวของ B

(AB)ᵢⱼ = Σₖ aᵢₖbₖⱼ
ตัวอย่าง: การคูณเมทริกซ์
A = ⎡ 1 2 ⎤ B = ⎡ 5 ⎤
⎣ 3 4 ⎦ ⎣ 6 ⎦

AB = ⎡ 1×5 + 2×6 ⎤ = ⎡ 17 ⎤
⎣ 3×5 + 4×6 ⎦ ⎣ 39 ⎦

3. ดีเทอร์มิแนนต์

ดีเทอร์มิแนนต์คำนวณได้สำหรับเมทริกซ์สี่เหลี่ยมจัตุรัสเท่านั้น ถ้าดีเทอร์มิแนนต์ของเมทริกซ์เป็นศูนย์ จะเรียกว่าดีเทอร์มิแนนต์เอกพจน์ และถ้าเป็นหนึ่ง จะเรียกว่ายูนิมอดูลาร์

ดีเทอร์มิแนนต์ขนาด 2×2

det(A) = |A| = ⎢ a b ⎥ = ad - bc
⎣ c d ⎦

ดีเทอร์มิแนนต์ขนาด 3×3

⎢ a₁ b₁ c₁ ⎥
⎢ a₂ b₂ c₂ ⎥ = a₁(b₂c₃ - b₃c₂) - b₁(a₂c₃ - a₃c₂) + c₁(a₂b₃ - a₃b₂)
⎣ a₃ b₃ c₃ ⎦
ตัวอย่าง: การหาดีเทอร์มิแนนต์ 2×2
A = ⎡ 3 2 ⎤
⎣ 1 4 ⎦

det(A) = (3)(4) - (2)(1) = 12 - 2 = 10

เครื่องคำนวณดีเทอร์มิแนนต์ 2×2

ผลลัพธ์จะแสดงที่นี่

4. เมทริกซ์ผกผัน

เมทริกซ์ผกผัน A⁻¹ มีอยู่ก็ต่อเมื่อดีเทอร์มิแนนต์ไม่เป็นศูนย์ เมื่อเมทริกซ์ผกผันคูณกับเมทริกซ์เดิมจะได้เมทริกซ์หนึ่ง

เงื่อนไขการมีอยู่ของเมทริกซ์ผกผัน

A⁻¹ มีอยู่ ⟺ det(A) ≠ 0
A × A⁻¹ = A⁻¹ × A = I

สูตรเมทริกซ์ผกผัน 2×2

A = ⎡ a b ⎤ A⁻¹ = (1/det(A)) ⎡ d -b ⎤
⎣ c d ⎦ ⎣ -c a ⎦
ตัวอย่าง: การหาเมทริกซ์ผกผัน
A = ⎡ 2 1 ⎤ det(A) = 2×3 - 1×1 = 5
⎣ 1 3 ⎦

A⁻¹ = (1/5) ⎡ 3 -1 ⎤ = ⎡ 3/5 -1/5 ⎤
⎣ -1 2 ⎦ ⎣-1/5 2/5 ⎦

การหาเมทริกซ์ผกผันด้วยการดำเนินการแถว

วิธีนี้ใช้การสร้าง augmented matrix [A|I] แล้วทำการลดรูปแถวจนได้ [I|A⁻¹]

5. การแก้ระบบสมการเชิงเส้นด้วยเมทริกซ์

รูปแบบเมทริกซ์ของระบบสมการ

AX = B

โดย A = เมทริกซ์สัมประสิทธิ์

X = เมทริกซ์ตัวแปร

B = เมทริกซ์คำตอบ

วิธีแก้:

X = A⁻¹B (ถ้า det(A) ≠ 0)

กฎของครามเมอร์

xᵢ = det(Aᵢ)/det(A)

โดย Aᵢ = เมทริกซ์ที่แทนหลักที่ i ด้วย B

ตัวอย่าง: ระบบสมการ 2 ตัวแปร

ระบบสมการ:

2x + y = 5

x + 3y = 8

A = ⎡ 2 1 ⎤ B = ⎡ 5 ⎤
⎣ 1 3 ⎦ ⎣ 8 ⎦

det(A) = 6 - 1 = 5

A₁ = ⎡ 5 1 ⎤ det(A₁) = 15 - 8 = 7
⎣ 8 3 ⎦

A₂ = ⎡ 2 5 ⎤ det(A₂) = 16 - 5 = 11
⎣ 1 8 ⎦

x = 7/5, y = 11/5

6. การประยุกต์ใช้เมทริกซ์

การแปลงเรขาคณิต

การหมุน

⎡ cos θ -sin θ ⎤
⎣ sin θ cos θ ⎦

การสะท้อน (แกน x)

⎡ 1 0 ⎤
⎣ 0 -1 ⎦

การขยาย/หด

⎡ sx 0 ⎤
⎣ 0 sy ⎦

การประยุกต์ในชีวิตจริง

ตัวอย่าง: ปัญหาการขนส่ง

โจทย์: บริษัทขนส่งมี 3 คลังสินค้าและ 2 ร้านค้า ต้นทุนการขนส่งต่อหน่วยแสดงในเมทริกซ์:

ต้นทุน = ⎡ 5 7 ⎤ (คลัง 1 → ร้าน 1,2)
⎢ 3 4 ⎥ (คลัง 2 → ร้าน 1,2)
⎣ 6 2 ⎦ (คลัง 3 → ร้าน 1,2)

ถ้าต้องขนส่ง [10, 15] หน่วยจากคลัง 2 ไปร้าน 1 และ 2 ตามลำดับ

ต้นทุนทั้งหมด = [10, 15] × [3, 4] = 30 + 60 = 90 บาท

ตัวอย่าง: การวิเคราะห์เครือข่าย

โจทย์: วิเคราะห์การไหลของข้อมูลในเครือข่ายคอมพิวเตอร์

เมทริกซ์ adjacency สำหรับกราฟ 3 โหนด:

A = ⎡ 0 1 1 ⎤
⎢ 1 0 1 ⎥
⎣ 0 1 0 ⎦

A² จะบอกจำนวนเส้นทางยาว 2 ขั้นตอนระหว่างโหนดต่าง ๆ

7. สมบัติสำคัญของเมทริกซ์

การบวก

A + B = B + A

(A + B) + C = A + (B + C)

A + O = A

การคูณ

A(BC) = (AB)C

A(B + C) = AB + AC

AI = IA = A

ทรานสโพส

(A + B)ᵀ = Aᵀ + Bᵀ

(AB)ᵀ = BᵀAᵀ

(Aᵀ)ᵀ = A

ดีเทอร์มิแนนต์

det(AB) = det(A)det(B)

det(Aᵀ) = det(A)

det(A⁻¹) = 1/det(A)

เมทริกซ์ผกผัน

(AB)⁻¹ = B⁻¹A⁻¹

(Aᵀ)⁻¹ = (A⁻¹)ᵀ

(A⁻¹)⁻¹ = A

ค่าลักษณะเฉพาะ

det(A - λI) = 0

Av = λv

λ = eigenvalue, v = eigenvector

9. สูตรสำคัญที่ต้องจำ

สูตรหลักของเมทริกซ์

การดำเนินการ สูตร เงื่อนไข
การบวก/ลบ (A ± B)ᵢⱼ = aᵢⱼ ± bᵢⱼ ขนาดเท่ากัน
การคูณ (AB)ᵢⱼ = Σₖ aᵢₖbₖⱼ หลัก A = แถว B
ดีเทอร์มิแนนต์ 2×2 ad - bc เมทริกซ์จัตุรัส
เมทริกซ์ผกผัน 2×2 (1/det(A)) [d -b; -c a] det(A) ≠ 0
ทรานสโพส (Aᵀ)ᵢⱼ = aⱼᵢ -

เทคนิคการจำ

เคล็ดลับการจำดีเทอร์มิแนนต์ 2×2

กฎ: "ทแยงมุมหลัก ลบ ทแยงมุมรอง"

⎡ a b ⎤ → a×d - b×c
⎣ c d ⎦
(↘) - (↙)
เคล็ดลับการคูณเมทริกซ์

จำง่าย ๆ: "แถว × หลัก"

สมาชิกแถว i หลัก j = แถว i ของ A คูณหลัก j ของ B
