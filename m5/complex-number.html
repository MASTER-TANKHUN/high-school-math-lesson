1. ความรู้เบื้องต้นเกี่ยวกับจำนวนเชิงซ้อน

นิยาม

จำนวนเชิงซ้อน (Complex Numbers) คือ จำนวนที่ประกอบด้วยส่วนจริงและส่วนจินตภาพ สามารถเขียนในรูป a + bi โดยที่ a และ b เป็นจำนวนจริง และ i เป็นหน่วยจินตภาพที่มีคุณสมบัติ i² = -1

เหตุผลที่ต้องใช้จำนวนเชิงซ้อน

จำนวนจริงไม่สามารถแก้สมการได้ครบทุกกรณี เช่น สมการ x² + 1 = 0 ไม่มีผลเฉลยในระบบจำนวนจริง จำนวนเชิงซ้อนช่วยให้เราแก้สมการได้อย่างครบถ้วน และมีบทบาทสำคัญในหลายสาขา

x² + 1 = 0
x² = -1
x = ±√(-1) = ±i
แผนภาพแสดงโครงสร้างของจำนวนเชิงซ้อน

หน่วยจินตภาพ i

คุณสมบัติของ i
  • i = √(-1)
  • i² = -1
  • i³ = i² · i = -1 · i = -i
  • i⁴ = i² · i² = (-1)(-1) = 1
  • i⁵ = i⁴ · i = 1 · i = i (รูปแบบวนซ้ำ)

2. รูปแบบการแทนจำนวนเชิงซ้อน

รูปแบบมาตรฐาน (Standard Form)

z = a + bi

โดยที่:

  • z = จำนวนเชิงซ้อน
  • a = ส่วนจริง (Real Part) เขียน Re(z) = a
  • b = ส่วนจินตภาพ (Imaginary Part) เขียน Im(z) = b
  • i = หน่วยจินตภาพ
ตัวอย่าง
  • 3 + 4i → Re(z) = 3, Im(z) = 4
  • -2 + 5i → Re(z) = -2, Im(z) = 5
  • 7 - 3i → Re(z) = 7, Im(z) = -3
  • 5 → Re(z) = 5, Im(z) = 0 (จำนวนจริง)
  • 6i → Re(z) = 0, Im(z) = 6 (จำนวนจินตภาพล้วน)

จำนวนเชิงซ้อนคอนจูเกต

นิยาม

จำนวนเชิงซ้อนคอนจูเกต (Complex Conjugate) ของ z = a + bi คือ z* = a - bi หรือเขียนแทนด้วย z̄

ถ้า z = a + bi
แล้ว z̄ = a - bi
สมบัติสำคัญ
  • z · z̄ = a² + b² (เป็นจำนวนจริงเสมอ)
  • (z₁ + z₂)̄ = z̄₁ + z̄₂
  • (z₁ · z₂)̄ = z̄₁ · z̄₂
  • z + z̄ = 2a (เป็นจำนวนจริง)
  • z - z̄ = 2bi (เป็นจำนวนจินตภาพล้วน)
แสดงความสัมพันธ์ของจำนวนเชิงซ้อนกับคอนจูเกต

โมดูลัสของจำนวนเชิงซ้อน

นิยาม

โมดูลัส (Modulus) หรือค่าสัมบูรณ์ของจำนวนเชิงซ้อน z = a + bi คือ |z| = √(a² + b²)

|z| = |a + bi| = √(a² + b²)
ตัวอย่างการหาโมดูลัส

1. |3 + 4i| = √(3² + 4²) = √(9 + 16) = √25 = 5

2. |-2 + 5i| = √((-2)² + 5²) = √(4 + 25) = √29

3. |6i| = √(0² + 6²) = √36 = 6

3. การดำเนินการจำนวนเชิงซ้อน

การบวก-ลบ

(a + bi) ± (c + di) = (a ± c) + (b ± d)i
วิธีการ
  1. บวกหรือลบส่วนจริงกับส่วนจริง
  2. บวกหรือลบส่วนจินตภาพกับส่วนจินตภาพ
  3. รวมผลลัพธ์เป็นจำนวนเชิงซ้อนใหม่
ตัวอย่าง

การบวก: (3 + 2i) + (1 + 4i) = 4 + 6i

การลบ: (5 + 7i) - (2 + 3i) = 3 + 4i

4. ระนาบเชิงซ้อน

ระนาบเชิงซ้อน (Complex Plane)

ระนาบพิกัดที่ใช้แทนจำนวนเชิงซ้อน โดยแกนนอน (แกน x) แทนส่วนจริง และแกนตั้ง (แกน y) แทนส่วนจินตภาพ เรียกอีกชื่อหนึ่งว่า "ระนาบอาร์กอง" (Argand Plane)

การแทนจุดในระนาบเชิงซ้อน

z = a + bi ↔ จุด (a, b)
ตัวอย่าง
  • z₁ = 3 + 4i → จุด (3, 4)
  • z₂ = -2 + i → จุด (-2, 1)
  • z₃ = 5 → จุด (5, 0) อยู่บนแกนจริง
  • z₄ = -3i → จุด (0, -3) อยู่บนแกนจินตภาพ

ความหมายทางเรขาคณิต

โมดูลัสและระยะทาง

|z| = ระยะทางจากจุดกำเนิดถึงจุด z

|z₁ - z₂| = ระยะทางระหว่างจุด z₁ และ z₂

การบวกเชิงเรขาคณิต

การบวกจำนวนเชิงซ้อนเทียบเท่ากับการบวกเวกเตอร์ในระนาบ

ระนาบเชิงซ้อนแสดงจุดและเวกเตอร์ต่างๆ

เซตของจำนวนเชิงซ้อนในระนาบ

ตัวอย่างเซตสำคัญ
  • วงกลมรัศมี r: |z| = r
  • ดิสก์รัศมี r: |z| ≤ r
  • แถบแนวนอน: a ≤ Im(z) ≤ b
  • แถบแนวตั้ง: a ≤ Re(z) ≤ b
  • ครึ่งระนาบขวา: Re(z) > 0
  • ครึ่งระนาบบน: Im(z) > 0

5. รูปแบบเชิงขั้ว

รูปแบบเชิงขั้ว (Polar Form)

การแทนจำนวนเชิงซ้อนด้วยโมดูลัส (r) และมุมอาร์กิวเมนต์ (θ)

z = r(cos θ + i sin θ) = re^(iθ)

การแปลงรูปแบบ

จากรูปมาตรฐานเป็นเชิงขั้ว

ถ้า z = a + bi แล้ว:

  • r = |z| = √(a² + b²)
  • θ = arctan(b/a) (ต้องพิจารณาจตุภาค)
จากเชิงขั้วเป็นรูปมาตรฐาน

ถ้า z = r(cos θ + i sin θ) แล้ว:

  • a = r cos θ
  • b = r sin θ
  • z = a + bi
ตัวอย่าง

แปลง z = 1 + i เป็นรูปเชิงขั้ว

r = √(1² + 1²) = √2

θ = arctan(1/1) = π/4

ดังนั้น z = √2(cos π/4 + i sin π/4)

สูตรออยเลอร์

สูตรออยเลอร์ (Euler's Formula)
e^(iθ) = cos θ + i sin θ
สมบัติสำคัญ
  • e^(iπ) = -1 (อัตลักษณ์ออยเลอร์)
  • e^(i·0) = 1
  • e^(i·π/2) = i
  • e^(i·2π) = 1
z = re^(iθ)
|z| = r
arg(z) = θ
การแทนจำนวนเชิงซ้อนในรูปแบบเชิงขั้วและความสัมพันธ์กับพิกัดคาร์ทีเซียน

การดำเนินการในรูปเชิงขั้ว

การคูณ
z₁z₂ = r₁r₂ · e^(i(θ₁ + θ₂))

คูณโมดูลัส, บวกมุม

การหาร
z₁/z₂ = (r₁/r₂) · e^(i(θ₁ - θ₂))

หารโมดูลัส, ลบมุม

ทฤษฎีบทเดอมัวร์ (De Moivre's Theorem)
z^n = r^n · e^(inθ) = r^n(cos nθ + i sin nθ)

สำหรับการยกกำลัง n ที่เป็นจำนวนเต็ม

6. การแก้สมการเชิงซ้อน

สมการกำลังสอง

az² + bz + c = 0

z = (-b ± √(b² - 4ac)) / 2a
ดิสคริมิแนนต์ในจำนวนเชิงซ้อน
  • Δ > 0: มีผลเฉลยจริง 2 ค่า
  • Δ = 0: มีผลเฉลยจริงเดียว
  • Δ < 0: มีผลเฉลยเชิงซ้อน 2 ค่า (คอนจูเกตกัน)
ตัวอย่าง

แก้ z² + 2z + 5 = 0

Δ = 2² - 4(1)(5) = 4 - 20 = -16

z = (-2 ± √(-16)) / 2 = (-2 ± 4i) / 2

z₁ = -1 + 2i, z₂ = -1 - 2i

รากที่ n ของจำนวนเชิงซ้อน

สูตรสำหรับหารากที่ n

ถ้า z = re^(iθ) แล้วรากที่ n ของ z คือ:

w_k = ⁿ√r · e^(i(θ + 2πk)/n)
โดยที่ k = 0, 1, 2, ..., n-1
ตัวอย่าง: รากที่ 3 ของ 8

8 = 8e^(i·0)

w₀ = ∛8 · e^(i·0/3) = 2

w₁ = ∛8 · e^(i·2π/3) = 2e^(i·2π/3) = -1 + i√3

w₂ = ∛8 · e^(i·4π/3) = 2e^(i·4π/3) = -1 - i√3

รากที่ n ของเอกภาพในระนาบเชิงซ้อน

รากของเอกภาพ

รากที่ n ของเอกภาพ

คือผลเฉลยของสมการ z^n = 1

ω_k = e^(i·2πk/n) = cos(2πk/n) + i sin(2πk/n)
โดยที่ k = 0, 1, 2, ..., n-1
สมบัติสำคัญ
  • รากทั้งหมดอยู่บนวงกลมหนึ่งหน่วย
  • รากต่างๆ แบ่งวงกลมออกเป็น n ส่วนเท่าๆ กัน
  • ผลคุณของรากทั้งหมด = (-1)^(n+1)
  • ผลบวกของรากทั้งหมด = 0 (สำหรับ n > 1)

7. การประยุกต์ใช้งาน

วิศวกรรมไฟฟ้า

กระแสไฟฟ้าสลับ (AC)

จำนวนเชิงซ้อนช่วยในการคำนวณกระแสและแรงดันไฟฟ้าสลับ โดยใช้:

  • ส่วนจริง: ส่วนประกอบแอมพลิจูด
  • ส่วนจินตภาพ: ส่วนประกอบเฟส
ตัวอย่าง

แรงดัน: V = 120∠30° = 120(cos 30° + i sin 30°)

= 120(√3/2 + i/2) = 60√3 + 60i โวลต์

ฟิสิกส์ควอนตัม

ฟังก์ชันคลื่น

จำนวนเชิงซ้อนเป็นพื้นฐานของกลศาสตร์ควอนตัม ใช้ในการอธิบาย:

  • ฟังก์ชันคลื่นของอนุภาค
  • ความน่าจะเป็นในการวัด
  • การแทรกสอดและการเลี้ยวเบน

การประมวลผลสัญญาณ

การแปลงฟูเรียร์

ใช้จำนวนเชิงซ้อนในการวิเคราะห์ความถี่ของสัญญาณ

F(ω) = ∫ f(t)e^(-iωt) dt

กราฟิกคอมพิวเตอร์

การหมุนในระนาบ 2D

การคูณด้วย e^(iθ) เป็นการหมุนมุม θ

ใช้ในการสร้างภาพเคลื่อนไหวและเกม

การประยุกต์ใช้จำนวนเชิงซ้อนในสาขาต่างๆ

8. ตัวอย่างโจทย์

ตัวอย่างที่ 1: การดำเนินการ

โจทย์: จงคำนวณ (3 + 2i)² - (1 - i)³

วิธีทำ:

หา (3 + 2i)²:

(3 + 2i)² = 9 + 12i + 4i² = 9 + 12i - 4 = 5 + 12i

หา (1 - i)³:

(1 - i)² = 1 - 2i + i² = 1 - 2i - 1 = -2i

(1 - i)³ = (1 - i)(-2i) = -2i + 2i² = -2i - 2 = -2 - 2i

คำตอบ: (5 + 12i) - (-2 - 2i) = 7 + 14i

ตัวอย่างที่ 2: รูปแบบเชิงขั้ว

โจทย์: จงแปลง z = -1 + i√3 เป็นรูปแบบเชิงขั้ว

วิธีทำ:

หาโมดูลัส:

r = |z| = √((-1)² + (√3)²) = √(1 + 3) = 2

หามุม:

tan θ = √3/(-1) = -√3

เนื่องจาก z อยู่ในจตุภาคที่ II: θ = π - π/3 = 2π/3

คำตอบ: z = 2(cos 2π/3 + i sin 2π/3) = 2e^(i2π/3)

ตัวอย่างที่ 3: การหารากที่ n

โจทย์: จงหารากที่ 4 ของ -16

วิธีทำ:

เขียนในรูปเชิงขั้ว:

-16 = 16e^(iπ)

ใช้สูตรรากที่ n:

w_k = ⁴√16 · e^(i(π + 2πk)/4) = 2e^(i(π + 2πk)/4)

สำหรับ k = 0, 1, 2, 3:

w₀ = 2e^(iπ/4) = 2(cos π/4 + i sin π/4) = √2 + i√2

w₁ = 2e^(i3π/4) = 2(cos 3π/4 + i sin 3π/4) = -√2 + i√2

w₂ = 2e^(i5π/4) = 2(cos 5π/4 + i sin 5π/4) = -√2 - i√2

w₃ = 2e^(i7π/4) = 2(cos 7π/4 + i sin 7π/4) = √2 - i√2

ตัวอย่างที่ 4: สมการเชิงซ้อน

โจทย์: แก้สมการ z² - (3 + i)z + (2 + 2i) = 0

วิธีทำ:

a = 1, b = -(3 + i), c = 2 + 2i

หาดิสคริมิแนนต์:

Δ = (3 + i)² - 4(1)(2 + 2i)

= 9 + 6i + i² - 8 - 8i

= 9 + 6i - 1 - 8 - 8i = -2i

หา √(-2i):

ให้ √(-2i) = x + yi

(x + yi)² = -2i

x² - y² + 2xyi = -2i

x² - y² = 0 และ 2xy = -2

จาก xy = -1 และ x² = y², ได้ x = ±1, y = ∓1

ดังนั้น √(-2i) = ±(1 - i)

หาผลเฉลย:

z = ((3 + i) ± (1 - i)) / 2

z₁ = (4)/2 = 2

z₂ = (2 + 2i)/2 = 1 + i

รวมตัวอย่างการแก้โจทย์จำนวนเชิงซ้อนแบบต่างๆ

เทคนิคการแก้โจทย์

เทคนิคสำคัญ
  1. จำรูปแบบของ i: i, -1, -i, 1 วนซ้ำทุก 4 ยกกำลัง
  2. ใช้คอนจูเกต: สำหรับการหารและทำให้ตัวส่วนเป็นจำนวนจริง
  3. แปลงรูปแบบ: เลือกรูปที่เหมาะสมกับโจทย์
  4. ใช้ทฤษฎีบทเดอมัวร์: สำหรับการยกกำลังและหาราก
  5. ตรวจสอบผลลัพธ์: แทนค่ากลับเข้าสมการเดิม
ข้อควรระวัง
  • ระวังเครื่องหมายเมื่อคำนวณ i²
  • พิจารณาจตุภาคเมื่อหามุมในรูปเชิงขั้ว
  • รากที่ n จะมี n ค่า (ไม่ใช่ค่าเดียว)
  • ตรวจสอบโดเมนของฟังก์ชันเชิงซ้อน

สรุปสำคัญ

จุดสำคัญที่ต้องจำ
  1. รูปแบบพื้นฐาน: z = a + bi โดย i² = -1
  2. การดำเนินการ: บวก-ลบแยกส่วน, คูณ-หารใช้คอนจูเกต
  3. ระนาบเชิงซ้อน: แกน x = ส่วนจริง, แกน y = ส่วนจินตภาพ
  4. รูปเชิงขั้ว: z = re^(iθ) เหมาะสำหรับการคูณ-หาร-ยกกำลัง
  5. โมดูลัส: |z| = ระยะทางจากจุดกำเนิด
  6. การประยุกต์: วิศวกรรม ฟิสิกส์ คอมพิวเตอร์
