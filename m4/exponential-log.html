1. ความรู้เบื้องต้นเกี่ยวกับฟังก์ชันเอกซ์โพเนนเชียล

นิยาม

ฟังก์ชันเอกซ์โพเนนเชียล (Exponential Function) คือ ฟังก์ชันที่มีรูปแบบ f(x) = a^x โดยที่ a > 0 และ a ≠ 1

ความสำคัญของฟังก์ชันเอกซ์โพเนนเชียล

ฟังก์ชันเอกซ์โพเนนเชียลมีความสำคัญในหลายด้าน:

  • การเติบโตของประชากร: แบบจำลองการเพิ่มจำนวนประชากร
  • การลงทุนและดอกเบิ้ย: การคำนวณดอกเบิ้ยทบต้น
  • การสลายตัวของสารกัมมันตรังสี: ครึ่งชีวิตของธาตุ
  • อุณหภูมิและการถ่ายเทความร้อน: กฎการเย็นตัวของนิวตัน
  • วิทยาการคอมพิวเตอร์: ความซับซ้อนของอัลกอริทึม
ตัวอย่างฟังก์ชันเอกซ์โพเนนเชียล
  • f(x) = 2^x (ฐาน 2)
  • f(x) = 3^x (ฐาน 3)
  • f(x) = e^x (ฐาน e ≈ 2.718)
  • f(x) = 10^x (ฐาน 10)
  • f(x) = (1/2)^x = 2^(-x) (ฟังก์ชันลดลง)

2. สมบัติของเลขยกกำลัง

กฎการคูณ
a^m × a^n = a^(m+n)
กฎการหาร
a^m ÷ a^n = a^(m-n)
กฎยกกำลัง
(a^m)^n = a^(mn)
กฎการคูณฐาน
(ab)^n = a^n × b^n
กฎการหารฐาน
(a/b)^n = a^n / b^n
เลขยกกำลังลบ
a^(-n) = 1/a^n

เลขยกกำลังเศส

a^(m/n) = ⁿ√(a^m) = (ⁿ√a)^m

a^0 = 1 (a ≠ 0)

a^1 = a
ตัวอย่าง
  • 8^(2/3) = ³√(8²) = ³√64 = 4
  • 27^(1/3) = ³√27 = 3
  • 16^(3/4) = ⁴√(16³) = ⁴√4096 = 8

3. ฟังก์ชันเอกซ์โพเนนเชียล

นิยามฟังก์ชันเอกซ์โพเนนเชียล

ฟังก์ชัน f(x) = a^x โดยที่ a > 0, a ≠ 1 และ x เป็นจำนวนจริงใดๆ

f(x) = a^x

โดเมน: (-∞, +∞)
เรนจ์: (0, +∞)

กรณี a > 1 (ฟังก์ชันเพิ่ม)

ลักษณะของกราฟ
  • ผ่านจุด (0, 1)
  • เป็นฟังก์ชันเพิ่มขึ้น
  • มีเส้นกำกับแนวนอน y = 0
  • เมื่อ x → +∞ แล้ว f(x) → +∞
  • เมื่อ x → -∞ แล้ว f(x) → 0

กรณี 0 < a < 1 (ฟังก์ชันลด)

ลักษณะของกราฟ
  • ผ่านจุด (0, 1)
  • เป็นฟังก์ชันลดลง
  • มีเส้นกำกับแนวนอน y = 0
  • เมื่อ x → +∞ แล้ว f(x) → 0
  • เมื่อ x → -∞ แล้ว f(x) → +∞

ฟังก์ชันเอกซ์โพเนนเชียลพิเศษ: e^x

จำนวน e (อีออยเลอร์)

e ≈ 2.71828... เป็นฐานธรรมชาติของฟังก์ชันเอกซ์โพเนนเชียล

e = lim(n→∞) (1 + 1/n)^n ≈ 2.71828
ความสำคัญของ e
  • ใช้ในการคำนวณดอกเบิ้ยทบต้นแบบต่อเนื่อง
  • ใช้ในฟิสิกส์และวิศวกรรมศาสตร์
  • ใช้ในสถิติและความน่าจะเป็น
  • เป็นฐานของลอการิทึมธรรมชาติ

4. ฟังก์ชันลอการิทึม

นิยามลอการิทึม

ลอการิทึมเป็นฟังก์ชันผกผันของฟังก์ชันเอกซ์โพเนนเชียล

y = log_a(x) ⟺ a^y = x

โดยที่ a > 0, a ≠ 1, x > 0

ลอการิทึมสามัญ

log(x) = log₁₀(x)

ลอการิทึมฐาน 10 ใช้ในการคำนวณทั่วไป

ลอการิทึมธรรมชาติ

ln(x) = log_e(x)

ลอการิทึมฐาน e ใช้ในแคลคูลัสและวิทยาศาสตร์

ลอการิทึมฐาน 2

log₂(x)

ใช้ในวิทยาการคอมพิวเตอร์และสารสนเทศ

ลักษณะของฟังก์ชันลอการิทึม

สมบัติของ f(x) = log_a(x)
  • โดเมน: (0, +∞)
  • เรนจ์: (-∞, +∞)
  • ผ่านจุด (1, 0)
  • มีเส้นกำกับแนวตั้ง x = 0
  • เป็นฟังก์ชันเพิ่มขึ้นเมื่อ a > 1
  • เป็นฟังก์ชันลดลงเมื่อ 0 < a < 1
ตัวอย่างการคำนวณลอการิทึม
  • log₂(8) = 3 เพราะ 2³ = 8
  • log₁₀(100) = 2 เพราะ 10² = 100
  • log₅(25) = 2 เพราะ 5² = 25
  • log₃(1/9) = -2 เพราะ 3⁻² = 1/9
  • ln(e) = 1 เพราะ e¹ = e

5. สมบัติของลอการิทึม

สมบัติการคูณ
log_a(mn) = log_a(m) + log_a(n)
สมบัติการหาร
log_a(m/n) = log_a(m) - log_a(n)
สมบัติยกกำลัง
log_a(m^n) = n × log_a(m)
การเปลี่ยนฐาน
log_a(x) = log_b(x) / log_b(a)
สมบัติพิเศษ
log_a(1) = 0
log_a(a) = 1
log_a(a^x) = x
a^(log_a(x)) = x
ลอการิทึมผกผัน
log_a(b) × log_b(a) = 1

log_a(b) = 1 / log_b(a)
ตัวอย่างการใช้สมบัติลอการิทึม

ตัวอย่างที่ 1: คำนวณ log₂(8 × 16)

log₂(8 × 16) = log₂(8) + log₂(16) = 3 + 4 = 7

ตัวอย่างที่ 2: คำนวณ log₃(27/9)

log₃(27/9) = log₃(27) - log₃(9) = 3 - 2 = 1

ตัวอย่างที่ 3: คำนวณ log₅(5³)

log₅(5³) = 3 × log₅(5) = 3 × 1 = 3

6. การแก้สมการเอกซ์โพเนนเชียลและลอการิทึม

การแก้สมการเอกซ์โพเนนเชียล

วิธีการแก้
  1. จัดรูปให้ฐานเท่ากัน (ถ้าทำได้)
  2. ถ้าฐานเท่ากัน เลขยกกำลังต้องเท่ากัน
  3. ใช้ลอการิทึมแก้ทั้งสองข้าง
  4. ใช้สมบัติของลอการิทึม
ตัวอย่าง

แก้สมการ: 2^(x+1) = 8

2^(x+1) = 2³

x + 1 = 3

x = 2

การแก้สมการลอการิทึม

วิธีการแก้
  1. กำหนดเงื่อนไขของลอการิทึม
  2. ใช้สมบัติของลอการิทึม
  3. เปลี่ยนเป็นรูปเอกซ์โพเนนเชียล
  4. ตรวจสอบคำตอบ
ตัวอย่าง

แก้สมการ: log₂(x) = 3

2³ = x

x = 8

ตรวจสอบ: log₂(8) = 3 ✓

สมการเอกซ์โพเนนเชียลซับซ้อน

ตัวอย่างที่ 1: ใช้ลอการิทึม

แก้สมการ: 3^x = 10

log(3^x) = log(10)

x × log(3) = 1

x = 1/log(3) ≈ 2.096

ตัวอย่างที่ 2: การแทนค่า

แก้สมการ: 4^x - 6(2^x) + 8 = 0

ให้ y = 2^x แล้ว 4^x = (2²)^x = (2^x)² = y²

y² - 6y + 8 = 0

(y - 2)(y - 4) = 0

y = 2 หรือ y = 4

2^x = 2 → x = 1 หรือ 2^x = 4 → x = 2

ข้อควรระวัง
  • ตรวจสอบโดเมนของลอการิทึม (x > 0)
  • ลอการิทึมของจำนวนลบไม่มีค่าจริง
  • ตรวจสอบคำตอบโดยแทนกลับในสมการเดิม

7. กราฟฟังก์ชัน

เปรียบเทียบกราฟฟังก์ชันเอกซ์โพเนนเชียลและลอการิทึม

ฟังก์ชันเอกซ์โพเนนเชียล y = 2^x
  • โดเมน: (-∞, +∞)
  • เรนจ์: (0, +∞)
  • ผ่านจุด (0, 1)
  • เส้นกำกับ: y = 0
  • เป็นฟังก์ชันเพิ่ม
ฟังก์ชันลอการิทึม y = log₂(x)
  • โดเมน: (0, +∞)
  • เรนจ์: (-∞, +∞)
  • ผ่านจุด (1, 0)
  • เส้นกำกับ: x = 0
  • เป็นฟังก์ชันเพิ่ม
ความสัมพันธ์ระหว่างฟังก์ชัน

ฟังก์ชันเอกซ์โพเนนเชียลและลอการิทึมเป็นฟังก์ชันผกผันกัน

  • กราฟสะท้อนกันผ่านเส้น y = x
  • โดเมนของฟังก์ชันหนึ่งเป็นเรนจ์ของอีกฟังก์ชันหนึ่ง
  • f(f⁻¹(x)) = x และ f⁻¹(f(x)) = x

8. การประยุกต์ใช้งาน

การเติบโตและการสลายตัว

N(t) = N₀ × e^(kt)

โดยที่:
N(t) = จำนวณ ณ เวลา t
N₀ = จำนวนเริ่มต้น
k = อัตราการเติบโต/สลายตัว
t = เวลา
ตัวอย่าง: การเติบโตของแบคทีเรีย

แบคทีเรียจำนวน 1000 ตัว เพิ่มขึ้น 50% ทุก 2 ชั่วโมง

จำนวนแบคทีเรียหลัง 6 ชั่วโมง = ?

N(6) = 1000 × (1.5)³ = 1000 × 3.375 = 3375 ตัว

ดอกเบิ้ยทบต้น

A = P(1 + r/n)^(nt)

สำหรับทบต้นต่อเนื่อง:
A = Pe^(rt)

โดยที่:
A = จำนวนเงินสุดท้าย
P = เงินต้น
r = อัตราดอกเบิ้ยต่อปี
n = จำนวนครั้งที่ทบต้นต่อปี
t = จำนวนปี
ตัวอย่าง: การลงทุน

ฝากเงิน 10,000 บาท ดอกเบิ้ย 5% ต่อปี ทบต้นรายปี

หลัง 10 ปี จะได้เงินเท่าไร?

A = 10000(1 + 0.05)¹⁰ = 10000 × 1.629 = 16,290 บาท

ครึ่งชีวิตและการสลายตัว

N(t) = N₀ × (1/2)^(t/T)

โดยที่ T = ครึ่งชีวิต
ตัวอย่าง: การสลายตัวของคาร์บอน-14

คาร์บอน-14 มีครึ่งชีวิต 5730 ปี

หากเริ่มต้นมี 100 กรัม หลัง 11460 ปีจะเหลือเท่าไร?

N(11460) = 100 × (1/2)^(11460/5730) = 100 × (1/2)² = 25 กรัม

การวัดความเข้มเสียงและแผ่นดินไหว

ระดับเสียง (dB) = 10 × log₁₀(I/I₀)

ความเข้มแผ่นดินไหว (Richter) = log₁₀(A/A₀)
ตัวอย่าง

เสียงปกติ 60 dB เปรียบเทียบกับเสียงเครื่องบิน 120 dB

เสียงเครื่องบินดังกว่ากี่เท่า?

120 - 60 = 60 dB = 10 × log₁₀(อัตราส่วน)

อัตราส่วน = 10^(60/10) = 10⁶ = 1,000,000 เท่า

pH และความเป็นกรด-ด่าง

pH = -log₁₀[H⁺]

โดยที่ [H⁺] = ความเข้มข้นของไฮโดรเจนไอออน
ตัวอย่าง

หาก [H⁺] = 1 × 10⁻³ M

pH = -log₁₀(1 × 10⁻³) = -(-3) = 3

สารนี้เป็นกรดแก่

9. ตัวอย่างโจทย์

ตัวอย่างที่ 1: การใช้สมบัติลอการิทึม

โจทย์: คำนวณ log₂(8) + log₂(4) - log₂(16)

วิธีทำ:

log₂(8) = log₂(2³) = 3

log₂(4) = log₂(2²) = 2

log₂(16) = log₂(2⁴) = 4

ดังนั้น 3 + 2 - 4 = 1

หรือใช้สมบัติ:

log₂(8 × 4 ÷ 16) = log₂(32/16) = log₂(2) = 1

ตัวอย่างที่ 2: แก้สมการเอกซ์โพเนนเชียล

โจทย์: แก้สมการ 5^(x-1) = 25

วิธีทำ:

5^(x-1) = 25

5^(x-1) = 5²

x - 1 = 2

x = 3

ตรวจสอบ: 5^(3-1) = 5² = 25 ✓

ตัวอย่างที่ 3: แก้สมการลอการิทึม

โจทย์: แก้สมการ log₃(x + 6) - log₃(x - 2) = 2

วิธีทำ:

log₃((x + 6)/(x - 2)) = 2

(x + 6)/(x - 2) = 3²

(x + 6)/(x - 2) = 9

x + 6 = 9(x - 2)

x + 6 = 9x - 18

24 = 8x

x = 3

ตรวจสอบเงื่อนไข: x > 2 ✓

ตัวอย่างที่ 4: โจทย์ประยุกต์

โจทย์: เงินฝาก 50,000 บาท ดอกเบิ้ย 6% ต่อปี ทบต้นรายเดือน จะเป็น 100,000 บาท ใช้เวลากี่ปี?

วิธีทำ:

ใช้สูตร A = P(1 + r/n)^(nt)

100,000 = 50,000(1 + 0.06/12)^(12t)

2 = (1.005)^(12t)

log(2) = 12t × log(1.005)

t = log(2)/(12 × log(1.005))

t ≈ 11.58 ปี
