สถิติ (Statistics)

บทเรียนสถิติขั้นสูงสำหรับมัธยมศึกษาปีที่ 6 ครอบคลุมการวิเคราะห์และแปลผลข้อมูลเชิงสถิติ

สารบัญเนื้อหา

1. ความรู้เบื้องต้นเกี่ยวกับสถิติ

นิยาม

สถิติ (Statistics) คือ วิชาความรู้ที่เกี่ยวกับการเก็บรวบรวม การจัดกระทำ การวิเคราะห์ การแปลความหมาย และการนำเสนอข้อมูล เพื่อใช้ในการตัดสินใจหรือแก้ไขปัญหาต่างๆ

ประชากร (Population)

กลุ่มของสมาชิกทั้งหมดที่เราสนใจจะศึกษา

ตัวอย่าง

นักเรียนทั้งหมดในโรงเรียน, ประชาชนในประเทศไทย

กลุ่มตัวอย่าง (Sample)

กลุ่มย่อยของประชากรที่เราเลือกมาศึกษา

ตัวอย่าง

นักเรียน 100 คนจากโรงเรียน, ประชาชน 1,000 คนจากการสุ่ม

ภาพรวมของการศึกษาสถิติ ม.6

ประเภทของข้อมูล

ข้อมูลเชิงคุณภาพ (Qualitative Data)
  • ข้อมูลที่ไม่สามารถวัดเป็นตัวเลขได้
  • เช่น เพศ, สี, อาชีพ, ระดับการศึกษา
  • แบ่งเป็น: Nominal และ Ordinal
ข้อมูลเชิงปริมาณ (Quantitative Data)
  • ข้อมูลที่วัดเป็นตัวเลขได้
  • เช่น อายุ, น้ำหนัก, ส่วนสูง, คะแนนสอบ
  • แบ่งเป็น: Discrete และ Continuous

2. การแจกแจงความถี่

ตารางแจกแจงความถี่

เป็นการจัดกลุ่มข้อมูลเพื่อแสดงจำนวนครั้งที่ค่าแต่ละค่าหรือช่วงค่าปรากฏในชุดข้อมูล

องค์ประกอบของตารางความถี่

องค์ประกอบ สัญลักษณ์ ความหมาย
ช่วงชั้น (Class Interval) - ช่วงของข้อมูลที่จัดกลุ่ม
ความถี่ (Frequency) f จำนวนข้อมูลในแต่ละช่วงชั้น
ความถี่สัมพัทธ์ rf f/n (สัดส่วนของแต่ละช่วง)
ความถี่สะสม cf ผลรวมของความถี่จากต้นจนถึงช่วงนั้น
ตัวอย่างตารางแจกแจงความถี่
ตัวอย่าง: การสร้างตารางความถี่

3. การแสดงข้อมูลด้วยกราฟ

ฮิสโตแกรม (Histogram)

กราฟแท่งที่แสดงการแจกแจงของข้อมูลต่อเนื่อง โดยแท่งจะติดกัน

แผนภูมิกล่อง (Box Plot)

กราฟที่แสดงการกระจายของข้อมูลผ่าน 5 ค่า: Min, Q₁, Q₂, Q₃, Max

แผนภาพกระจาย (Scatter Plot)

กราฟที่แสดงความสัมพันธ์ระหว่างตัวแปร 2 ตัว

ตัวอย่างฮิสโตแกรมแสดงการแจกแจงของข้อมูล

4. ค่าสถิติและค่าสถิติถ่วงน้ำหนัก

ค่าเฉลี่ย (Mean)
x̄ = Σx / n
หรือ x̄ = Σfx / Σf (สำหรับข้อมูลจัดกลุ่ม)

ค่ากลางของข้อมูลทั้งหมด

ค่ามัธยฐาน (Median)
Me = ค่ากลางเมื่อเรียงลำดับ
หรือ (n+1)/2 สำหรับตำแหน่ง

ค่าที่อยู่ตรงกลางเมื่อเรียงข้อมูลจากน้อยไปมาก

ฐานนิยม (Mode)
Mo = ค่าที่ปรากฏบ่อยที่สุด

ค่าที่มีความถี่สูงสุดในชุดข้อมูล

ค่าเฉลี่ยถ่วงน้ำหนัก (Weighted Mean)

x̄w = (w₁x₁ + w₂x₂ + ... + wₙxₙ) / (w₁ + w₂ + ... + wₙ)
ตัวอย่าง

คะแนนของนักเรียนคนหนึ่ง:

  • การบ้าน: 85 คะแนน (น้ำหนัก 20%)
  • สอบกลางภาค: 78 คะแนน (น้ำหนัก 30%)
  • สอบปลายภาค: 92 คะแนน (น้ำหนัก 50%)

คะแนนเฉลี่ยถ่วงน้ำหนัก = (0.2×85 + 0.3×78 + 0.5×92) = 86.4 คะแนน

การเปรียบเทียบค่าเฉลี่ย มัธยฐาน และฐานนิยม

5. การวัดการกระจาย

พิสัย (Range)
R = ค่าสูงสุด - ค่าต่ำสุด

ความแตกต่างระหว่างค่าสูงสุดและต่ำสุด

พิสัยควอไทล์ (IQR)
IQR = Q₃ - Q₁

ความแตกต่างระหว่างควอไทล์ที่ 3 และควอไทล์ที่ 1

ส่วนเบี่ยงเบนมาตรฐาน (Standard Deviation)

สำหรับประชากร:
σ = √(Σ(x-μ)² / N)
สำหรับกลุ่มตัวอย่าง:
s = √(Σ(x-x̄)² / (n-1))
ตัวอย่าง: การคำนวณส่วนเบี่ยงเบนมาตรฐาน

ข้อมูล: 2, 4, 6, 8, 10

x̄ = (2+4+6+8+10)/5 = 6

6. ควอไทล์และเปอร์เซนไทล์

ควอไทล์ (Quartiles)

ค่าที่แบ่งข้อมูลออกเป็น 4 ส่วนเท่าๆ กัน ประกอบด้วย Q₁, Q₂ (มัธยฐาน), Q₃

ควอไทล์ที่ 1 (Q₁)

ตำแหน่ง = (n+1)/4

แบ่งข้อมูล 25% แรก

ควอไทล์ที่ 2 (Q₂)

ตำแหน่ง = 2(n+1)/4

เท่ากับมัธยฐาน

ควอไทล์ที่ 3 (Q₃)

ตำแหน่ง = 3(n+1)/4

แบ่งข้อมูล 75% แรก

เปอร์เซนไทล์ (Percentiles)

ตำแหน่งของ Pₖ = k(n+1)/100

ค่าที่แบ่งข้อมูลออกเป็น 100 ส่วนเท่าๆ กัน

ตัวอย่าง

ข้อมูล: 12, 15, 18, 20, 22, 25, 28, 30, 32, 35 (n=10)

  • Q₁: ตำแหน่ง = (10+1)/4 = 2.75 → Q₁ = 15 + 0.75(18-15) = 17.25
  • Q₂: ตำแหน่ง = 2(10+1)/4 = 5.5 → Q₂ = (22+25)/2 = 23.5
  • Q₃: ตำแหน่ง = 3(10+1)/4 = 8.25 → Q₃ = 30 + 0.25(32-30) = 30.5
แผนภาพกล่องแสดงควอไทล์และค่าผิดปกติ

7. การวิเคราะห์ข้อมูล

การหาค่าผิดปกติ (Outliers)

ค่าต่ำผิดปกติ: < Q₁ - 1.5×IQR
ค่าสูงผิดปกติ: > Q₃ + 1.5×IQR
ข้อควรระวัง

ค่าผิดปกติอาจเกิดจากข้อผิดพลาดในการเก็บข้อมูล หรืออาจเป็นข้อมูลที่มีคุณค่าพิเศษ ต้องพิจารณาอย่างรอบคอบ

การแจกแจงแบบสมมาตร
  • Mean ≈ Median ≈ Mode
  • ข้อมูลกระจายเท่าๆ กันทั้งสองด้าน
  • เช่น การแจกแจงปกติ (Normal Distribution)
การแจกแจงเบ้
  • เบ้ซ้าย: Mean < Median < Mode
  • เบ้ขวา: Mode < Median < Mean
  • หางยาวไปทางด้านที่เบ้

8. ตัวอย่างโจทย์และแบบฝึกหัด

โจทย์ที่ 1: การคำนวณค่าสถิติ

คะแนนสอบคณิตศาสตร์ของนักเรียน 15 คน ได้ดังนี้:

68, 72, 75, 69, 81, 77, 73, 80, 76, 78, 71, 74, 79, 70, 82

วิธีทำ

1. เรียงข้อมูลจากน้อยไปมาก:

68, 69, 70, 71, 72, 73, 74, 75, 76, 77, 78, 79, 80, 81, 82

2. หาค่าเฉลี่ย:

x̄ = (68+69+...+82)/15 = 1125/15 = 75

3. หามัธยฐาน:

ตำแหน่งที่ 8 → Me = 75

4. หาควอไทล์:

Q₁ = 71, Q₂ = 75, Q₃ = 79

IQR = 79 - 71 = 8

โจทย์ที่ 2: การหาค่าผิดปกติ

จากข้อมูลเดิม เพิ่มคะแนน 95 และ 45 เข้าไป จงหาค่าผิดปกติ

วิธีทำ

ขั้นตอนที่ 1: คำนวณขอบเขตค่าผิดปกติ

ขอบล่าง = Q₁ - 1.5×IQR = 71 - 1.5×8 = 59

ขอบบน = Q₃ + 1.5×IQR = 79 + 1.5×8 = 91

ขั้นตอนที่ 2: ตรวจสอบค่าผิดปกติ

45 < 59 → 45 เป็นค่าผิดปกติต่ำ

95 > 91 → 95 เป็นค่าผิดปกติสูง
