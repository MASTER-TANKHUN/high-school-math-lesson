1. ความรู้เบื้องต้นเกี่ยวกับฟังก์ชัน

นิยาม

ฟังก์ชัน (Function) หมายถึง ความสัมพันธ์ (x, y) ใดๆ โดยที่ ถ้าตัวหน้าเหมือนกัน ตัวหลังจะต้องเหมือนกัน

หรืออีกนัยหนึ่ง สมาชิกแต่ละตัวในโดเมนจะจับคู่กับสมาชิกในเรนจ์เพียงหนึ่งตัวเท่านั้น

การตรวจสอบฟังก์ชัน

สำหรับการตรวจสอบว่ากราฟหนึ่งเป็นฟังก์ชันหรือไม่ สามารถใช้

วิธีเส้นตั้งฉาก (Vertical Line Test)

ลากเส้นขนานกับแกน y หากเส้นดังกล่าวตัดกับกราฟได้เพียงจุดเดียวทุกจุด แสดงว่ากราฟนั้นเป็นฟังก์ชัน

หากตัดกับกราหได้มากกว่าหนึ่งจุด แสดงว่าไม่เป็นฟังก์ชัน

การเขียนฟังก์ชัน

ฟังก์ชันสามารถเขียนได้หลายรูปแบบ:

y = f(x)
f: A → B
{(x, y) | y = f(x), x ∈ A}

โดยที่ x เป็นตัวแปรอิสระ (Independent Variable) และ y เป็นตัวแปรตาม (Dependent Variable)

2. โดเมนและเรนจ์

โดเมน (Domain)

โดเมนของฟังก์ชัน f คือ เซตของค่าทั้งหมดของ x ที่ทำให้ f(x) มีความหมาย

เขียนแทนด้วย Dom(f) หรือ Df

ตัวอย่าง

f(x) = √(x-2)

โดเมน: x - 2 ≥ 0

ดังนั้น Dom(f) = [2, +∞)

เรนจ์ (Range)

เรนจ์ของฟังก์ชัน f คือ เซตของค่าทั้งหมดของ f(x) หรือ y ที่เป็นไปได้

เขียนแทนด้วย Ran(f) หรือ Rf

ตัวอย่าง

f(x) = x² + 1

เนื่องจาก x² ≥ 0 สำหรับทุกค่า x

ดังนั้น Ran(f) = [1, +∞)

การหาโดเมนและเรนจ์

ขั้นตอนการหาโดเมน
  1. หาเงื่อนไขที่ทำให้ฟังก์ชันไม่มีความหมาย
  2. ตัวส่วนต้องไม่เท่ากับศูนย์
  3. ในเครื่องหมายรากที่สอง ค่าใต้รากต้องไม่เป็นลบ
  4. ในฟังก์ชันลอการิทึม ค่าในลอการิทึมต้องเป็นบวก
  5. เขียนคำตอบเป็นช่วง (Interval Notation)
ขั้นตอนการหาเรนจ์
  1. วิเคราะห์พฤติกรรมของฟังก์ชัน
  2. หาจุดสูงสุดและต่ำสุด (ถ้ามี)
  3. ดูการเปลี่ยนแปลงของค่า y
  4. ใช้กราฟช่วยในการวิเคราะห์
  5. เขียนคำตอบเป็นช่วง

3. ประเภทของฟังก์ชัน

ฟังก์ชันหนึ่งต่อหนึ่ง (One-to-One)

สมาชิกแต่ละตัวในเรนจ์จะมีต้นภาพเพียงหนึ่งตัวในโดเมน

ถ้า f(a) = f(b) แล้ว a = b
ฟังก์ชันทั่วถึง (Onto)

สมาชิกทุกตัวในโคโดเมนจะมีต้นภาพในโดเมน

เรนจ์ = โคโดเมน
ฟังก์ชันสองทาง (Bijective)

เป็นทั้งฟังก์ชันหนึ่งต่อหนึ่งและทั่วถึง

f: A → B เป็น bijective

การตรวจสอบประเภทฟังก์ชัน

วิธีเส้นนอน (Horizontal Line Test)

สำหรับตรวจสอบฟังก์ชันหนึ่งต่อหนึ่ง:

ลากเส้นขนานกับแกน x หากเส้นตัดกับกราฟได้เพียงจุดเดียวทุกจุด แสดงว่าเป็นฟังก์ชันหนึ่งต่อหนึ่ง

4. ฟังก์ชันประกอบ

นิยาม

ฟังก์ชันประกอบ (Composite Function) คือ การนำเอาผลลัพธ์ของฟังก์ชันหนึ่งไปเป็นค่านำเข้าของอีกฟังก์ชันหนึ่ง

(f ∘ g)(x) = f(g(x))

อ่านว่า "f ประกอบ g ของ x"

ขั้นตอนการหาฟังก์ชันประกอบ
  1. ใส่ g(x) แทนที่ x ในฟังก์ชัน f
  2. ทำการคำนวณและจัดรูป
  3. หาโดเมนของฟังก์ชันประกอบ
  4. ตรวจสอบผลลัพธ์
ตัวอย่างที่ 1

กำหนด: f(x) = x² + 1, g(x) = 2x - 3

หา: (f ∘ g)(x)

วิธีทำ:

(f ∘ g)(x) = f(g(x)) = f(2x - 3)

= (2x - 3)² + 1

= 4x² - 12x + 9 + 1

= 4x² - 12x + 10

ข้อควรระวัง

โดยทั่วไปแล้ว f ∘ g ≠ g ∘ f

การประกอบฟังก์ชันไม่มีสมบัติการสลับที่

ตัวอย่างที่ 2

จากตัวอย่างที่ 1 หา: (g ∘ f)(x)

(g ∘ f)(x) = g(f(x)) = g(x² + 1)

= 2(x² + 1) - 3

= 2x² + 2 - 3

= 2x² - 1

สังเกต: (f ∘ g)(x) ≠ (g ∘ f)(x)

โดเมนของฟังก์ชันประกอบ

การหาโดเมนของ (f ∘ g)(x)

โดเมนของ f ∘ g คือ เซตของค่า x ที่:

  1. x อยู่ในโดเมนของ g
  2. g(x) อยู่ในโดเมนของ f
Dom(f ∘ g) = {x ∈ Dom(g) | g(x) ∈ Dom(f)}

5. ฟังก์ชันผกผัน

นิยาม

ฟังก์ชันผกผัน (Inverse Function) ของฟังก์ชัน f คือ ฟังก์ชัน f⁻¹ ที่มีสมบัติ

f⁻¹(f(x)) = x สำหรับทุก x ∈ Dom(f)
f(f⁻¹(x)) = x สำหรับทุก x ∈ Dom(f⁻¹)
เงื่อนไขการมีฟังก์ชันผกผัน

ฟังก์ชัน f จะมีฟังก์ชันผกผัน f⁻¹ ก็ต่อเมื่อ f เป็นฟังก์ชันหนึ่งต่อหนึ่ง (Injective)

ใช้เส้นนอนตรวจสอบได้

วิธีหาฟังก์ชันผกผัน
  1. เขียน y = f(x)
  2. สลับตำแหน่ง x กับ y
  3. แก้หา y ใหม่
  4. เขียน y = f⁻¹(x)
ตัวอย่างที่ 1

หาฟังก์ชันผกผันของ: f(x) = 3x - 2

วิธีทำ:

1) y = 3x - 2

2) x = 3y - 2

3) x + 2 = 3y

y = (x + 2)/3

4) f⁻¹(x) = (x + 2)/3

สมบัติของฟังก์ชันผกผัน
  • Dom(f⁻¹) = Ran(f)
  • Ran(f⁻¹) = Dom(f)
  • กราฟของ f และ f⁻¹ สมมาตรกันผ่านเส้น y = x
  • (f⁻¹)⁻¹ = f

6. การเลื่อนกราฟ

การเลื่อนแนวตั้ง (Vertical Shifts)

กฎการเลื่อนแนวตั้ง
y = f(x) + k

• ถ้า k > 0: เลื่อนขึ้น k หน่วย
• ถ้า k < 0: เลื่อนลง |k| หน่วย

การเลื่อนแนวนอน (Horizontal Shifts)

กฎการเลื่อนแนวนอน
y = f(x - h)

• ถ้า h > 0: เลื่อนไปขวา h หน่วย
• ถ้า h < 0: เลื่อนไปซ้าย |h| หน่วย

การย่อขยาย

การย่อขยายแนวตั้ง
y = af(x)

• |a| > 1: ขยายแนวตั้ง
• 0 < |a| < 1: หดแนวตั้ง
• a < 0: พลิกผ่านแกน x
การย่อขยายแนวนอน
y = f(bx)

• |b| > 1: หดแนวนอน
• 0 < |b| < 1: ขยายแนวนอน
• b < 0: พลิกผ่านแกน y
ตัวอย่างการเลื่อนกราฟ

กำหนด: f(x) = x²

การเลื่อน:

  • g(x) = x² + 3 (เลื่อนขึ้น 3 หน่วย)
  • h(x) = (x - 2)² (เลื่อนขวา 2 หน่วย)
  • k(x) = (x + 1)² - 4 (เลื่อนซ้าย 1, ลง 4)
  • m(x) = 2x² (ขยายแนวตั้ง 2 เท่า)
  • n(x) = (2x)² (หดแนวนอน 1/2 เท่า)

การสาธิตการเลื่อนกราหแบบโต้ตอบ

สมการปัจจุบัน: y = f(x)

ตัวอย่างการเลื่อนกราฟฟังก์ชันต่างๆ

7. ฟังก์ชันพหุนาม

นิยาม

ฟังก์ชันพหุนาม (Polynomial Function) คือ ฟังก์ชันที่มีรูปแบบ

P(x) = aₙxⁿ + aₙ₋₁xⁿ⁻¹ + ... + a₁x + a₀

โดยที่ aᵢ เป็นค่าคงที่ และ n เป็นจำนวนเต็มไม่ลบ

ฟังก์ชันเชิงเส้น (Linear)
f(x) = ax + b
(a ≠ 0)

กราฟเป็นเส้นตรง

ความชัน = a

จุดตัดแกน y = (0, b)

ฟังก์ชันกำลังสอง (Quadratic)
f(x) = ax² + bx + c
(a ≠ 0)

กราฟเป็นพาราโบลา

จุดยอด = (-b/2a, f(-b/2a))

เปิดขึ้นถ้า a > 0

ฟังก์ชันกำลังสาม (Cubic)
f(x) = ax³ + bx² + cx + d
(a ≠ 0)

กราฟมีรูปร่างคล้าย S

สามารถมีจุดเปลี่ยนทิศได้สูงสุด 2 จุด

พฤติกรรมของฟังก์ชันพหุนาม

End Behavior

พฤติกรรมของกราฟเมื่อ x → ±∞ ขึ้นอยู่กับ:

  1. ดีกรีของพหุนาม (n)
  2. เครื่องหมายของสัมประสิทธิ์หน้า (aₙ)
กฎ End Behavior
  • n คู่, aₙ > 0: ขึ้นทั้งสองข้าง
  • n คู่, aₙ < 0: ลงทั้งสองข้าง
  • n คี่, aₙ > 0: ลงซ้าย, ขึ้นขวา
  • n คี่, aₙ < 0: ขึ้นซ้าย, ลงขวา

8. ฟังก์ชันตรรกยะ

นิยาม

ฟังก์ชันตรรกยะ (Rational Function) คือ ฟังก์ชันที่เขียนได้ในรูป

f(x) = P(x)/Q(x)

โดยที่ P(x) และ Q(x) เป็นฟังก์ชันพหุนาม และ Q(x) ≠ 0

เส้นกรรม (Asymptotes)

เส้นกรรมแนวตั้ง

เกิดเมื่อตัวส่วน = 0 แต่ตัวเศษ ≠ 0

x = a เมื่อ Q(a) = 0
และ P(a) ≠ 0
เส้นกรรมแนวนอน

ขึ้นอยู่กับดีกรีของตัวเศษและตัวส่วน

• deg(P) < deg(Q): y = 0
• deg(P) = deg(Q): y = aₙ/bₙ
• deg(P) > deg(Q): ไม่มี
เส้นกรรมเอียง

เกิดเมื่อ deg(P) = deg(Q) + 1

y = mx + b
หาจากการหารยาว
ตัวอย่างที่ 1

วิเคราะห์: f(x) = (x + 1)/(x - 2)

โดเมน: x ≠ 2

เส้นกรรมแนวตั้ง: x = 2

เส้นกรรมแนวนอน: y = 1

จุดตัดแกน x: x = -1

จุดตัดแกน y: y = -1/2

ขั้นตอนการวิเคราะห์ฟังก์ชันตรรกยะ
  1. หาโดเมน
  2. หาเส้นกรรมแนวตั้ง
  3. หาเส้นกรรมแนวนอนหรือเอียง
  4. หาจุดตัดแกน
  5. ตรวจสอบพฤติกรรมรอบเส้นกรรม
  6. เขียนกราฟ
กราฟฟังก์ชันตรรกยะพร้อมเส้นกรรม

9. ฟังก์ชันเอกซ์โพเนนเชียล

นิยาม

ฟังก์ชันเอกซ์โพเนนเชียล (Exponential Function) คือ ฟังก์ชันที่มีรูปแบบ

f(x) = aˣ

โดยที่ a > 0 และ a ≠ 1

สมบัติของฟังก์ชันเอกซ์โพเนนเชียล
  • โดเมน: (-∞, +∞)
  • เรนจ์: (0, +∞)
  • ผ่านจุด (0, 1)
  • เส้นกรรมแนวนอน: y = 0
  • เป็นฟังก์ชันหนึ่งต่อหนึ่ง
กฎการคำนวณ
  • aˣ · aʸ = aˣ⁺ʸ
  • aˣ ÷ aʸ = aˣ⁻ʸ
  • (aˣ)ʸ = aˣʸ
  • a⁰ = 1
  • a⁻ˣ = 1/aˣ
พฤติกรรมของกราฟ

เมื่อ a > 1:

  • ฟังก์ชันเพิ่มขึ้น
  • เมื่อ x → +∞, f(x) → +∞
  • เมื่อ x → -∞, f(x) → 0

เมื่อ 0 < a < 1:

  • ฟังก์ชันลดลง
  • เมื่อ x → +∞, f(x) → 0
  • เมื่อ x → -∞, f(x) → +∞

ฟังก์ชันเอกซ์โพเนนเชียลธรรมชาติ

ฟังก์ชัน f(x) = eˣ

โดยที่ e ≈ 2.71828... เป็นจำนวนออยเลอร์

เป็นฟังก์ชันเอกซ์โพเนนเชียลที่สำคัญที่สุดในคณิตศาสตร์

e = lim(n→∞) (1 + 1/n)ⁿ
การประยุกต์ใช้
  • การเติบโตของประชากร: P(t) = P₀e^(kt)
  • การสลายตัวกัมมันตรังสี: N(t) = N₀e^(-λt)
  • ดอกเบียวทบเชิงซ้อน: A = Pe^(rt)
  • การระบายความร้อน: T(t) = T₀e^(-kt)
กราฟฟังก์ชันเอกซ์โพเนนเชียลสำหรับค่า a ต่างๆ

10. ฟังก์ชันลอการิทึม

นิยาม

ฟังก์ชันลอการิทึม (Logarithmic Function) คือ ฟังก์ชันผกผันของฟังก์ชันเอกซ์โพเนนเชียล

f(x) = log_a(x)

y = log_a(x) ⟺ aʸ = x

โดยที่ a > 0, a ≠ 1, และ x > 0

สมบัติของฟังก์ชันลอการิทึม
  • โดเมน: (0, +∞)
  • เรนจ์: (-∞, +∞)
  • ผ่านจุด (1, 0)
  • เส้นกรรมแนวตั้ง: x = 0
  • เป็นฟังก์ชันหนึ่งต่อหนึ่ง
กฎการคำนวณลอการิทึม
  • log_a(xy) = log_a(x) + log_a(y)
  • log_a(x/y) = log_a(x) - log_a(y)
  • log_a(xⁿ) = n·log_a(x)
  • log_a(1) = 0
  • log_a(a) = 1
  • log_a(1/x) = -log_a(x)
ลอการิทึมพิเศษ
ลอการิทึมสามัญ (Common Logarithm)
log(x) = log₁₀(x)

ฐาน 10

ลอการิทึมธรรมชาติ (Natural Logarithm)
ln(x) = log_e(x)

ฐาน e

การเปลี่ยนฐาน

log_a(x) = log_b(x) / log_b(a) = ln(x) / ln(a)
ตัวอย่าง

หาค่าของ: log₃(27)

วิธีที่ 1: ใช้นิยาม

log₃(27) = y ⟺ 3ʸ = 27 = 3³

ดังนั้น y = 3

วิธีที่ 2: เปลี่ยนฐาน

log₃(27) = ln(27) / ln(3) = ln(3³) / ln(3) = 3ln(3) / ln(3) = 3

กราฟฟังก์ชันลอการิทึมและเอกซ์โพเนนเชียลเป็นฟังก์ชันผกผันกัน

11. ตัวอย่างโจทย์

ตัวอย่างที่ 1: โดเมนและเรนจ์

โจทย์: หาโดเมนและเรนจ์ของ f(x) = √(4 - x²)

วิธีทำ:

หาโดเมน:

4 - x² ≥ 0

x² ≤ 4

-2 ≤ x ≤ 2

Dom(f) = [-2, 2]

หาเรนจ์:

เมื่อ x = 0: f(0) = √4 = 2 (ค่าสูงสุด)

เมื่อ x = ±2: f(±2) = √0 = 0 (ค่าต่ำสุด)

Ran(f) = [0, 2]

ตัวอย่างที่ 2: ฟังก์ชันประกอบ

โจทย์: กำหนด f(x) = 2x + 1, g(x) = x² - 3
หา (f ∘ g)(x) และ (g ∘ f)(x)

วิธีทำ:

(f ∘ g)(x) = f(g(x))

= f(x² - 3)

= 2(x² - 3) + 1

= 2x² - 6 + 1

= 2x² - 5

(g ∘ f)(x) = g(f(x))

= g(2x + 1)

= (2x + 1)² - 3

= 4x² + 4x + 1 - 3

= 4x² + 4x - 2

ตัวอย่างที่ 3: ฟังก์ชันผกผัน

โจทย์: หาฟังก์ชันผกผันของ f(x) = (3x - 2)/(x + 1)

วิธีทำ:

1) ตั้ง y = (3x - 2)/(x + 1)

2) สลับ x กับ y: x = (3y - 2)/(y + 1)

3) แก้หา y:

x(y + 1) = 3y - 2

xy + x = 3y - 2

xy - 3y = -x - 2

y(x - 3) = -x - 2

y = (-x - 2)/(x - 3)

ดังนั้น: f⁻¹(x) = (-x - 2)/(x - 3)

ตัวอย่างที่ 4: ฟังก์ชันเอกซ์โพเนนเชียล

โจทย์: แก้สมการ 2^(x+1) = 8^(x-2)

วิธีทำ:

2^(x+1) = 8^(x-2)

2^(x+1) = (2³)^(x-2)

2^(x+1) = 2^(3(x-2))

2^(x+1) = 2^(3x-6)

เปรียบเทียบเลขชี้กำลัง:

x + 1 = 3x - 6

7 = 2x

x = 7/2

ตัวอย่างที่ 5: ฟังก์ชันลอการิทึม

โจทย์: แก้สมการ log₂(x + 3) + log₂(x - 1) = 3

วิธีทำ:

log₂(x + 3) + log₂(x - 1) = 3

log₂[(x + 3)(x - 1)] = 3

(x + 3)(x - 1) = 2³ = 8

x² + 2x - 3 = 8

x² + 2x - 11 = 0

ใช้สูตรกำลังสอง:

x = (-2 ± √(4 + 44))/2 = (-2 ± √48)/2 = (-2 ± 4√3)/2 = -1 ± 2√3

ตรวจสอบเงื่อนไข: x > 1

ดังนั้น x = -1 + 2√3

ตัวอย่างที่ 6: การเลื่อนกราฟ

โจทย์: อธิบายการเปลี่ยนแปลงของกราฟ g(x) = -2f(x - 3) + 1
เมื่อเทียบกับ f(x) = x²

วิธีทำ:

จาก g(x) = -2f(x - 3) + 1 เมื่อ f(x) = x²

g(x) = -2(x - 3)² + 1

การเปลี่ยนแปลง:

  1. x - 3: เลื่อนขวา 3 หน่วย
  2. -2: ขยาย 2 เท่าและพลิกผ่านแกน x
  3. +1: เลื่อนขึ้น 1 หน่วย

จุดยอดใหม่: (3, 1)

เปิด: ลง (เพราะ a = -2 < 0)

รวมตัวอย่างการแก้โจทย์ฟังก์ชันประเภทต่างๆ
