1. ความรู้เบื้องต้นเกี่ยวกับจำนวนจริง

นิยาม

จำนวนจริง (Real Numbers) คือ จำนวนที่สามารถแสดงบนเส้นจำนวนได้ทั้งหมด ประกอบด้วยจำนวนตรรกยะและจำนวนอตรรกยะ

ระบบจำนวนจริง

ระบบจำนวนจริงเป็นระบบที่ประกอบด้วย:

  • เซตของจำนวนจริง (R)
  • ความสัมพันธ์ระหว่างสมาชิกของเซตจำนวนจริง
  • การดำเนินการบวก (+) และการคูณ (×)
  • สมบัติเกี่ยวกับความจริง
แผนภาพแสดงระบบจำนวนจริง

2. การจำแนกจำนวนจริง

จำนวนตรรกยะ (Rational Numbers)

นิยาม

จำนวนที่เขียนในรูป a

b ได้ โดยที่ a และ b เป็นจำนวนเต็ม และ b ≠ 0

ตัวอย่าง
  • จำนวนเต็ม: ..., -2, -1, 0, 1, 2, ...
  • เศษส่วน: 1
    2    , 3
    4    , -5
    7
  • ทศนิยมจำกัด: 0.5, 0.75, -2.25
  • ทศนิยมซ้ำ: 0.333..., 1.272727...

จำนวนอตรรกยะ (Irrational Numbers)

นิยาม

จำนวนจริงที่ไม่เป็นจำนวนตรรกยะ ไม่สามารถเขียนในรูปเศษส่วนได้

ตัวอย่าง
  • √2, √3, √5 (รากที่สองที่ไม่ใช่กำลังสองสมบูรณ์)
  • π (ไพ) ≈ 3.141592...
  • e (อีออยเลอร์) ≈ 2.71828...
  • ทศนิยมไม่จำกัดไม่ซ้ำ
แผนภาพการจำแนกประเภทของจำนวนจริง

ความสัมพันธ์ระหว่างเซตของจำนวน

สัญลักษณ์ที่ใช้แทนเซตของจำนวน:

  • N = จำนวนธรรมชาติ = {1, 2, 3, 4, ...}
  • W = จำนวนนับ = {0, 1, 2, 3, 4, ...}
  • Z = จำนวนเต็ม = {..., -2, -1, 0, 1, 2, ...}
  • Q = จำนวนตรรกยะ
  • I = จำนวนอตรรกยะ
  • R = จำนวนจริง = Q ∪ I
N ⊂ W ⊂ Z ⊂ Q ⊂ R
I ⊂ R
Q ∩ I = ∅

3. สมบัติของจำนวนจริง

สมบัติปิด (Closure Property)

การบวก: a + b ∈ R

การคูณ: a × b ∈ R

สมบัติการสลับที่ (Commutative)

การบวก: a + b = b + a

การคูณ: a × b = b × a

สมบัติการเปลี่ยนหมู่ (Associative)

การบวก: (a + b) + c = a + (b + c)

การคูณ: (a × b) × c = a × (b × c)

สมบัติการมีเอกลักษณ์ (Identity)

การบวก: a + 0 = a

การคูณ: a × 1 = a

สมบัติการมีอินเวอร์ส (Inverse)

การบวก: a + (-a) = 0

การคูณ: a × (1/a) = 1, a ≠ 0

สมบัติการแจกแจง (Distributive)

a × (b + c) = (a × b) + (a × c)

(a + b) × c = (a × c) + (b × c)

4. การหารพหุนาม

ขั้นตอนการหารพหุนาม
ตัวตั้ง = ตัวหาร × ผลหาร + เศษ
P(x) = Q(x) × T(x) + R(x)

โดยที่ดีกรีของเศษ R(x) น้อยกว่าดีกรีของตัวหาร Q(x)

การหารยาว (Long Division)

ตัวอย่างการหารพหุนามแบบยาว

การหารสังเคราะห์ (Synthetic Division)

ตัวอย่างการหารสังเคราะห์

ทฤษฎีบทเศษเหลือ (Remainder Theorem)

ทฤษฎีบทเศษเหลือ

เมื่อหารพหุนาม P(x) ด้วย (x - a) จะได้เศษเหลือเท่ากับ P(a)

P(x) = (x - a) × Q(x) + P(a)
การใช้ทฤษฎีบทเศษเหลือในการหาเศษ

5. การแยกตัวประกอบ

ขั้นตอนการแยกตัวประกอบพหุนาม
  1. หาตัวประกอบของ an (สัมประสิทธิ์ของ xn) และ a0 (พจน์คงที่)
  2. หาค่า k/m ที่ทำให้ P(k/m) = 0
  3. ใช้การหารสังเคราะห์หาร P(x) ด้วย (x - k/m)
  4. นำผลหารไปแยกตัวประกอบต่อไปจนได้ตัวประกอบทั้งหมด
สมบัติการแยกตัวประกอบพหุนาม
ตัวอย่างการแยกตัวประกอบ
ตัวอย่างขั้นตอนการแยกตัวประกอบพหุนาม

6. การแก้สมการและอสมการ

การแก้สมการพหุนามดีกรี 2

ax² + bx + c = 0

x = (-b ± √(b² - 4ac)) / 2a
วิธีการแก้สมการดีกรี 2
ดิสคริมิแนนต์ (Discriminant)

Δ = b² - 4ac

  • Δ > 0: มีคำตอบจริง 2 คำตอบ
  • Δ = 0: มีคำตอบจริงเดียว
  • Δ < 0: ไม่มีคำตอบจริง

การแก้สมการพหุนามดีกรีสูง

ขั้นตอนการแก้
  1. แยกตัวประกอบพหุนาม
  2. ใช้หลักการ ab = 0 แล้ว a = 0 หรือ b = 0
  3. แก้สมการเชิงเส้นหรือกำลังสองที่ได้
ตัวอย่างการแก้สมการดีกรีสูง

การแก้อสมการพหุนาม

ขั้นตอนการแก้อสมการ
  1. จัดรูปอสมการให้ด้านหนึ่งเท่ากับ 0
  2. แยกตัวประกอบ และทำให้สัมประสิทธิ์หน้าเป็นบวก
  3. หาค่าตัวแปรที่ทำให้แต่ละตัวประกอบเท่ากับ 0
  4. วาดเส้นจำนวนและใส่เครื่องหมาย + - + - สลับกัน
  5. เลือกช่วงที่ตรงตามเงื่อนไขของโจทย์
ข้อควรระวัง

เมื่อคูณหรือหารด้วยจำนวนลบ ต้องพลิกเครื่องหมายอสมการ

7. ค่าสัมบูรณ์

นิยามค่าสัมบูรณ์
|x| = { x เมื่อ x ≥ 0
{ -x เมื่อ x < 0

สมบัติของค่าสัมบูรณ์

สมบัติสำคัญของค่าสัมบูรณ์
สมบัติพื้นฐาน
  • |x| ≥ 0
  • |x| = |-x|
  • |xy| = |x||y|
  • |x/y| = |x|/|y|, y ≠ 0
  • |x + y| ≤ |x| + |y| (อสมการสามเหลี่ยม)

กราฟค่าสัมบูรณ์

รูปแบบกราฟฟังก์ชันค่าสัมบูรณ์

การแก้สมการและอสมการค่าสัมบูรณ์

วิธีการแก้สมการและอสมการค่าสัมบูรณ์
หลักการแก้สมการค่าสัมบูรณ์
|f(x)| = a (a > 0) ⟺ f(x) = a หรือ f(x) = -a

|f(x)| < a (a > 0) ⟺ -a < f(x) < a

|f(x)| > a (a > 0) ⟺ f(x) > a หรือ f(x) < -a

สมบัติการไม่เท่ากัน

สมบัติสำคัญของการไม่เท่ากันในจำนวนจริง

8. ตัวอย่างโจทย์

ตัวอย่างที่ 1: การแยกตัวประกอบ

โจทย์: จงแยกตัวประกอบของ P(x) = x³ - 6x² + 11x - 6

วิธีทำ:

  1. หาตัวประกอบของ 6: ±1, ±2, ±3, ±6
  2. ทดสอบ P(1) = 1 - 6 + 11 - 6 = 0
  3. ดังนั้น (x - 1) เป็นตัวประกอบ
  4. หาร P(x) ด้วย (x - 1) ได้ x² - 5x + 6
  5. แยกตัวประกอบต่อ: x² - 5x + 6 = (x - 2)(x - 3)

คำตอบ: P(x) = (x - 1)(x - 2)(x - 3)

ตัวอย่างที่ 2: การแก้อสมการ

โจทย์: แก้อสมการ x³ - 4x² + x + 6 > 0

วิธีทำ:

  1. แยกตัวประกอบ: (x + 1)(x - 2)(x - 3) > 0
  2. หาจุดตัด: x = -1, 2, 3
  3. ทดสอบเครื่องหมาย:
    • x < -1: (-)(-)(-) = - (ลบ)
    • -1 < x < 2: (+)(-)(-) = + (บวก)
    • 2 < x < 3: (+)(+)(-) = - (ลบ)
    • x > 3: (+)(+)(+) = + (บวก)

คำตอบ: x ∈ (-1, 2) ∪ (3, +∞)

ตัวอย่างที่ 3: ค่าสัมบูรณ์

โจทย์: แก้สมการ |2x - 3| = 7

วิธีทำ:

จาก |2x - 3| = 7 ได้

2x - 3 = 7 หรือ 2x - 3 = -7

กรณีที่ 1: 2x - 3 = 7

2x = 10

x = 5

กรณีที่ 2: 2x - 3 = -7

2x = -4

x = -2

คำตอบ: x = 5 หรือ x = -2

ตัวอย่างที่ 4: อสมการค่าสัมบูรณ์

โจทย์: แก้อสมการ |x - 2| < 5

วิธีทำ:

จาก |x - 2| < 5 ได้

-5 < x - 2 < 5

-5 + 2 < x < 5 + 2

-3 < x < 7

คำตอบ: x ∈ (-3, 7)

รวมโจทย์จำนวนจริงพร้อมเฉลยละเอียด

แบบทดสอบความเข้าใจ

  • ก) 0.25
  • ข) 22/7
  • ค) √7
  • ง) 0.333...
ข้อ 2: ถ้า |x - 3| = 4 แล้ว x เท่ากับข้อใด?
  • ก) 1 เท่านั้น
  • ข) 7 เท่านั้น
  • ค) -1 หรือ 7
  • ง) 1 หรือ -7
ข้อ 3: เมื่อหาร x³ - 2x² + 3x - 6 ด้วย (x - 2) จะได้เศษเท่าไร?
  • ก) 0
  • ข) 2
  • ค) 4
  • ง) -2
ข้อ 4: ผลเฉลยของอสมการ |2x + 1| ≤ 5 คือข้อใด?
  • ก) x ≤ -3 หรือ x ≥ 2
  • ข) -3 ≤ x ≤ 2
  • ค) x < -3 หรือ x > 2
  • ง) -2 ≤ x ≤ 3
แบบฝึकหัดเพิ่มเติมสำหรับฝึกทักษะ

แบบฝึกหัดเพิ่มเติม

ข้อ 5: จงแยกตัวประกอบของ P(x) = x⁴ - 10x² + 9

ข้อ 6: แก้อสมการ (x - 1)(x + 2)(x - 3) ≥ 0

ข้อ 7: หาผลเฉลยของสมการ |x² - 4| = 5

ข้อ 8: พิสูจน์ว่า √2 + √3 เป็นจำนวนอตรรกยะ

ข้อ 9: แก้อสมการ |x - 1| + |x + 2| < 5

ข้อ 10: หาค่า a เพื่อให้สมการ x² - 2ax + a² - 1 = 0 มีผลเฉลยเป็นจำนวนจริง

เฉลยแบบฝึกหัดเพิ่มเติม

ข้อ 5: P(x) = (x² - 1)(x² - 9) = (x - 1)(x + 1)(x - 3)(x + 3)

ข้อ 6: x ∈ [-2, 1] ∪ [3, +∞)

ข้อ 7: x = ±3 หรือ x = ±1

ข้อ 9: x ∈ (-2, 3)

ข้อ 10: a เป็นจำนวนจริงใดๆ (เพราะ Δ = 4 > 0 เสมอ)
