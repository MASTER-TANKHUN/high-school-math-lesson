แคลคูลัสเบื้องต้น

บทเรียนพื้นฐานที่สำคัญสำหรับคณิตศาสตร์ ม.6 และการเรียนต่อในระดับมหาวิทยาลัย

🎯 บทนำ

แคลคูลัสเป็นสาขาหนึ่งของคณิตศาสตร์ที่ศึกษาเกี่ยวกับการเปลี่ยนแปลงและการสะสม ในบทนี้เราจะเรียนรู้เกี่ยวกับ ลิมิต อนุพันธ์ และปริพันธ์ ซึ่งเป็นพื้นฐานสำคัญสำหรับการศึกษาในระดับมหาวิทยาลัย โดยเฉพาะในสาขาวิทยาศาสตร์ วิศวกรรมศาสตร์ เศรษฐศาสตร์ และการแพทย์

📊 ลิมิตของฟังก์ชัน (Limits)

ลิมิตคือค่าที่ฟังก์ชัน f(x) เข้าใกล้เมื่อ x เข้าใกล้ค่าใดค่าหนึ่ง ลิมิตเป็นแนวคิดพื้นฐานที่สำคัญในการทำความเข้าใจอนุพันธ์และปริพันธ์

สัญลักษณ์ลิมิต

$$\lim_{x \to a} f(x) = L$$

หมายความว่า เมื่อ x เข้าใกล้ a แล้ว f(x) จะเข้าใกล้ L

กฎสำคัญของลิมิต:

  • $\lim_{x \to a} [f(x) + g(x)] = \lim_{x \to a} f(x) + \lim_{x \to a} g(x)$
  • $\lim_{x \to a} [f(x) \cdot g(x)] = \lim_{x \to a} f(x) \cdot \lim_{x \to a} g(x)$
  • $\lim_{x \to a} \frac{f(x)}{g(x)} = \frac{\lim_{x \to a} f(x)}{\lim_{x \to a} g(x)}$ (เมื่อ $\lim_{x \to a} g(x) \neq 0$)
ตัวอย่าง: การหาลิมิต

หาค่า $\lim_{x \to 2} \frac{x^2 - 4}{x - 2}$

วิธีทำ: แยกตัวประกอบ $x^2 - 4 = (x-2)(x+2)$

$\lim_{x \to 2} \frac{(x-2)(x+2)}{x-2} = \lim_{x \to 2} (x+2) = 4$

🔗 ความต่อเนื่องของฟังก์ชัน (Continuity)

ฟังก์ชัน f(x) จะต่อเนื่องที่ x = a เมื่อเงื่อนไขสามข้อต่อไปนี้เป็นจริง

เงื่อนไขความต่อเนื่อง

1. f(a) มีค่าจำกัด (f(a) ถูกนิยาม)

2. $\lim_{x \to a} f(x)$ มีค่าจำกัด

3. $\lim_{x \to a} f(x) = f(a)$

หมายเหตุ:

ฟังก์ชันที่ต่อเนื่องจะมีกราฟที่เป็นเส้นเดียวกันโดยไม่มีการขาด หรือการกระโดด

📐 อนุพันธ์ของฟังก์ชัน (Derivatives)

อนุพันธ์แสดงถึงอัตราการเปลี่ยนแปลงของฟังก์ชัน ณ จุดใดจุดหนึ่ง และสามารถใช้หาความชันของเส้นสัมผัสได้

นิยามอนุพันธ์

$$f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$$

สูตรอนุพันธ์พื้นฐาน:

$\frac{d}{dx}(c) = 0$

$\frac{d}{dx}(x^n) = nx^{n-1}$

$\frac{d}{dx}(e^x) = e^x$

$\frac{d}{dx}(\ln x) = \frac{1}{x}$

$\frac{d}{dx}(\sin x) = \cos x$

$\frac{d}{dx}(\cos x) = -\sin x$

$\frac{d}{dx}(\tan x) = \sec^2 x$

$\frac{d}{dx}(a^x) = a^x \ln a$

ตัวอย่าง: การหาอนุพันธ์

หาอนุพันธ์ของ $f(x) = 3x^4 - 2x^3 + 5x - 1$

คำตอบ: $f'(x) = 12x^3 - 6x^2 + 5$

🎯 การประยุกต์ของอนุพันธ์

1. การหาค่าสูงสุดและต่ำสุด

ใช้หลักการที่ว่า ณ จุดสูงสุดหรือต่ำสุด อนุพันธ์จะมีค่าเท่ากับศูนย์
ขั้นตอนการหาค่าสูงสุด-ต่ำสุด

1. หา f'(x) = 0 เพื่อหาจุดวิกฤต

2. ทดสอบด้วยอนุพันธ์อันดับสอง f''(x)

• ถ้า f''(x) > 0 → ค่าต่ำสุด

• ถ้า f''(x) < 0 → ค่าสูงสุด

2. ความชันของเส้นสัมผัส

ความชันของเส้นสัมผัสของกราฟ y = f(x) ณ จุด (a, f(a)) คือ f'(a)
ตัวอย่าง: การประยุกต์ใช้

กล่องสี่เหลี่ยมผืนผ้าที่มีความยาวรอบรูป 20 เซนติเมตร จะมีพื้นที่สูงสุดเท่าใด?

วิธีทำ: ให้ x เป็นความกว้าง แล้วความยาว = 10-x

พื้นที่ A = x(10-x) = 10x - x²

A' = 10 - 2x = 0 → x = 5

ดังนั้น พื้นที่สูงสุด = 5×5 = 25 ตารางเซนติเมตร

∫ ปฏิยานุพันธ์ (Antiderivatives & Integration)

ปฏิยานุพันธ์เป็นกระบวนการตรงข้ามกับการหาอนุพันธ์ หาก F'(x) = f(x) แล้ว F(x) จะเป็นปฏิยานุพันธ์ของ f(x)

สัญลักษณ์ปริพันธ์

$$\int f(x) dx = F(x) + C$$

โดย F'(x) = f(x) และ C คือค่าคงที่

สูตรปริพันธ์พื้นฐาน:

$\int x^n dx = \frac{x^{n+1}}{n+1} + C$

$\int e^x dx = e^x + C$

$\int \frac{1}{x} dx = \ln|x| + C$

$\int a^x dx = \frac{a^x}{\ln a} + C$

$\int \sin x dx = -\cos x + C$

$\int \cos x dx = \sin x + C$

$\int \sec^2 x dx = \tan x + C$

$\int \frac{1}{\sqrt{1-x^2}} dx = \arcsin x + C$

ปริพันธ์จำกัดเขต

$$\int_a^b f(x) dx = F(b) - F(a)$$

โดย F'(x) = f(x)

📏 พื้นที่ปิดล้อมด้วยเส้นโค้ง

ปริพันธ์จำกัดเขตสามารถใช้คำนวณพื้นที่ใต้เส้นโค้งได้ ซึ่งมีประโยชน์ในการแก้ปัญหาจริงมากมาย

การหาพื้นที่:

พื้นที่ระหว่างเส้นโค้งกับแกน x

$$A = \int_a^b |f(x)| dx$$

พื้นที่ระหว่างสองเส้นโค้ง

$$A = \int_a^b |f(x) - g(x)| dx$$

ตัวอย่าง: การหาพื้นที่

หาพื้นที่ใต้เส้นโค้ง y = x² ระหว่าง x = 0 ถึง x = 2

วิธีทำ:

$A = \int_0^2 x^2 dx = \left[\frac{x^3}{3}\right]_0^2 = \frac{8}{3} - 0 = \frac{8}{3}$ ตารางหน่วย

📝 สรุปสำคัญ

จุดสำคัญที่ต้องจำ:
  • ลิมิต: พื้นฐานของแคลคูลัส ใช้หาความต่อเนื่องและอนุพันธ์
  • อนุพันธ์: แสดงอัตราการเปลี่ยนแปลง ใช้หาค่าสูงสุด-ต่ำสุด
  • ปริพันธ์: กระบวนการตรงข้ามอนุพันธ์ ใช้หาพื้นที่และปริมาตร
  • การประยุกต์: สามารถนำไปใช้ในฟิสิกส์ เศรษฐศาสตร์ และวิศวกรรม
