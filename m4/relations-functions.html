1. ความรู้เบื้องต้นเกี่ยวกับความสัมพันธ์

นิยาม

ความสัมพันธ์ (Relation) คือ การเชื่อมโยงสมาชิกของเซต A กับสมาชิกของเซต B โดยใช้กฎหรือเงื่อนไขใดๆ

การแสดงความสัมพันธ์

ความสัมพันธ์สามารถแสดงได้หลายวิธี:

  • คู่อันดับ (Ordered Pairs): R = {(a,b) | a ∈ A, b ∈ B}
  • แผนภาพลูกศร (Arrow Diagram): เส้นลูกศรเชื่อมระหว่างสมาชิก
  • ตารางสองทิศทาง: แสดงความสัมพันธ์ในรูปตาราง
  • กราฟบนระนาบพิกัด: จุดบนระนาบแสดงคู่อันดับ
ตัวอย่าง

ให้ A = {1, 2, 3} และ B = {4, 5, 6}

ความสัมพันธ์ R: "x น้อยกว่า y"

R = {(1,4), (1,5), (1,6), (2,5), (2,6), (3,6)}

2. ประเภทของความสัมพันธ์

ความสัมพันธ์สะท้อน (Reflexive)

สำหรับทุก a ∈ A แล้ว (a,a) ∈ R

ตัวอย่าง: ความสัมพันธ์ "เท่ากับ" หรือ "น้อยกว่าหรือเท่ากับ"

ความสัมพันธ์สมมาตร (Symmetric)

ถ้า (a,b) ∈ R แล้ว (b,a) ∈ R

ตัวอย่าง: ความสัมพันธ์ "เป็นพี่น้องกัน"

ความสัมพันธ์ปรรตรรก (Transitive)

ถ้า (a,b) ∈ R และ (b,c) ∈ R แล้ว (a,c) ∈ R

ตัวอย่าง: ความสัมพันธ์ "น้อยกว่า"

ความสัมพันธ์สมมูล (Equivalence)

ความสัมพันธ์ที่มีสมบัติสะท้อน สมมาตร และปรรตรรก

ตัวอย่าง: ความสัมพันธ์ "เท่ากับ"

3. ความรู้เบื้องต้นเกี่ยวกับฟังก์ชัน

นิยามฟังก์ชัน

ฟังก์ชัน (Function) คือ ความสัมพันธ์พิเศษจากเซต A ไปยังเซต B ที่สมาชิกทุกตัวใน A จับคู่กับสมาชิกใน B ได้เพียงตัวเดียว

f: A → B หรือ y = f(x)

เงื่อนไขของฟังก์ชัน

  • สมาชิกทุกตัวใน A ต้องจับคู่กับสมาชิกใน B
  • สมาชิกแต่ละตัวใน A จับคู่กับสมาชิกใน B ได้เพียงตัวเดียว
  • สมาชิกใน B อาจไม่ถูกจับคู่เลย หรือถูกจับคู่หลายครั้ง

การทดสอบฟังก์ชัน

Vertical Line Test

เส้นตรงดิ่งใดๆ ที่ลากผ่านกราฟต้องตัดกราฟได้มากที่สุด 1 จุด

ตัวอย่าง

เป็นฟังก์ชัน: f(x) = 2x + 1

ไม่เป็นฟังก์ชัน: x² + y² = 25 (วงกลม)

4. การเขียนฟังก์ชันและการหาค่า

สัญกรณ์ฟังก์ชัน

f(x) = นิพจน์ในตัวแปร x

ตัวอย่าง: f(x) = x² - 3x + 2
การหาค่าฟังก์ชัน

เพื่อหา f(a) ให้แทนค่า x = a ในนิพจน์ฟังก์ชัน

ตัวอย่าง: ถ้า f(x) = x² - 3x + 2

f(2) = (2)² - 3(2) + 2 = 4 - 6 + 2 = 0

ตัวอย่างการหาค่าฟังก์ชัน

ให้ f(x) = 2x - 1 และ g(x) = x² + 3

จงหา:

  • f(3) = 2(3) - 1 = 5
  • g(-2) = (-2)² + 3 = 7
  • f(a+1) = 2(a+1) - 1 = 2a + 1
ฟังก์ชันแบบเป็นชิ้น
f(x) = { x + 2 ถ้า x < 0
{ x² ถ้า x ≥ 0

f(-1) = (-1) + 2 = 1

f(2) = 2² = 4

5. โดเมนและเรนจ์

โดเมน (Domain)

เซตของค่า x ทั้งหมดที่ทำให้ฟังก์ชัน f(x) มีความหมาย

สัญลักษณ์: Dom(f) หรือ Df

เรนจ์ (Range)

เซตของค่า y ทั้งหมดที่เป็นไปได้จากฟังก์ชัน f(x)

สัญลักษณ์: Range(f) หรือ Rf

การหาโดเมน

กรณีที่ต้องระวัง
  • เศษส่วน: ตัวส่วนต้องไม่เท่ากับ 0
  • รากที่สอง: นิพจน์ใต้รากต้องไม่น้อยกว่า 0
  • ลอการิทึม: นิพจน์ในลอการิทึมต้องมากกว่า 0
ตัวอย่างการหาโดเมน

1. f(x) = 1/(x-2)

โดเมน: x ≠ 2 หรือ (-∞,2) ∪ (2,+∞)

2. f(x) = √(x-3)

x - 3 ≥ 0 → x ≥ 3

โดเมน: [3,+∞)

3. f(x) = √(x-1)/(x-4)

x - 1 ≥ 0 และ x ≠ 4

โดเมน: [1,4) ∪ (4,+∞)

6. ประเภทของฟังก์ชัน

ฟังก์ชันเชิงเส้น (Linear Function)
f(x) = ax + b

กราฟเป็นเส้นตรง

ตัวอย่าง: f(x) = 2x - 3

ฟังก์ชันกำลังสอง (Quadratic Function)
f(x) = ax² + bx + c

กราฟเป็นพาราโบลา

ตัวอย่าง: f(x) = x² - 4x + 3

ฟังก์ชันค่าสัมบูรณ์ (Absolute Function)
f(x) = |x|

กราฟเป็น V-shape

ตัวอย่าง: f(x) = |x - 2|

ฟังก์ชันคงที่ (Constant Function)
f(x) = c

กราฟเป็นเส้นตรงขนานแกน x

ตัวอย่าง: f(x) = 5

ฟังก์ชันตัวตน (Identity Function)
f(x) = x

กราฟเป็นเส้นตรงผ่าน (0,0) ทำมุม 45°

ฟังก์ชันรากที่สอง (Square Root Function)
f(x) = √x

โดเมน: x ≥ 0

เรนจ์: y ≥ 0

7. กราฟของฟังก์ชัน

การเขียนกราฟฟังก์ชัน

ขั้นตอนการเขียนกราฟ
  1. หาโดเมนและเรนจ์ของฟังก์ชัน
  2. หาจุดตัดแกน x และแกน y
  3. หาจุดสำคัญ (จุดสูงสุด จุดต่ำสุด)
  4. สร้างตารางค่า
  5. เขียนกราฟโดยเชื่อมจุดต่างๆ

การเลื่อนกราฟ

y = f(x) + k → เลื่อนแนวตั้ง k หน่วย
y = f(x - h) → เลื่อนแนวนอน h หน่วย
y = f(x - h) + k → เลื่อนทั้งสองทิศทาง

การสะท้อนกราฟ

y = -f(x) → สะท้อนผ่านแกน x
y = f(-x) → สะท้อนผ่านแกน y
y = -f(-x) → สะท้อนผ่านจุดกำเนิด
ตัวอย่างการเลื่อนกราฟ

ฟังก์ชันพื้นฐาน: f(x) = x²

  • g(x) = x² + 2 → เลื่อนขึ้น 2 หน่วย
  • h(x) = (x - 3)² → เลื่อนไปทางขวา 3 หน่วย
  • k(x) = (x - 1)² + 2 → เลื่อนไปทางขวา 1 หน่วยและขึ้น 2 หน่วย

8. ฟังก์ชันประกอบ

นิยามฟังก์ชันประกอบ

ฟังก์ชันประกอบ (Composite Function) ของ f และ g คือ

(f ∘ g)(x) = f(g(x))
(g ∘ f)(x) = g(f(x))
ขั้นตอนการหาฟังก์ชันประกอบ
  1. ระบุฟังก์ชัน f(x) และ g(x)
  2. แทนค่า g(x) ลงใน f(x) แทนที่ x
  3. ปรับรูปนิพจน์ให้เรียบง่าย
  4. หาโดเมนของฟังก์ชันประกอบ
ตัวอย่างฟังก์ชันประกอบ

ให้ f(x) = 2x + 1 และ g(x) = x² - 3

หา (f ∘ g)(x):

(f ∘ g)(x) = f(g(x)) = f(x² - 3)

= 2(x² - 3) + 1 = 2x² - 6 + 1 = 2x² - 5

หา (g ∘ f)(x):

(g ∘ f)(x) = g(f(x)) = g(2x + 1)

= (2x + 1)² - 3 = 4x² + 4x + 1 - 3 = 4x² + 4x - 2

ข้อควรระวัง

โดยทั่วไป (f ∘ g)(x) ≠ (g ∘ f)(x)

การประกอบฟังก์ชันไม่มีสมบัติการสลับที่

9. ฟังก์ชันผกผัน

นิยามฟังก์ชันผกผัน

ฟังก์ชัน f⁻¹ เป็นฟังก์ชันผกผันของ f ถ้า

f(f⁻¹(x)) = x และ f⁻¹(f(x)) = x
วิธีหาฟังก์ชันผกผัน
  1. เขียน y = f(x)
  2. สลับ x และ y ได้ x = f(y)
  3. แก้สมการหา y ในรูป y = f⁻¹(x)
  4. ตรวจสอบว่าเป็นฟังก์ชัน (ใช้ Horizontal Line Test)
ตัวอย่างการหาฟังก์ชันผกผัน

ให้ f(x) = 2x - 3

ขั้นที่ 1: y = 2x - 3

ขั้นที่ 2: x = 2y - 3

ขั้นที่ 3: 2y = x + 3

y = (x + 3)/2

ดังนั้น f⁻¹(x) = (x + 3)/2

สมบัติของฟังก์ชันผกผัน
  • กราฟของ f และ f⁻¹ สะท้อนกันผ่านเส้น y = x
  • Dom(f) = Range(f⁻¹)
  • Range(f) = Dom(f⁻¹)
  • (f⁻¹)⁻¹ = f

การทดสอบว่าฟังก์ชันมีฟังก์ชันผกผัน

Horizontal Line Test

เส้นตรงแนวนอนใดๆ ที่ลากผ่านกราฟต้องตัดกราฟได้มากที่สุด 1 จุด

หรือฟังก์ชันต้องเป็น One-to-One Function

10. ตัวอย่างโจทย์

ตัวอย่างที่ 1: การหาโดเมน

โจทย์: จงหาโดเมนของ f(x) = √(x² - 9)/(x - 2)

วิธีทำ:

เงื่อนไข:

  • x² - 9 ≥ 0
  • x - 2 ≠ 0

จาก x² - 9 ≥ 0 ได้ (x-3)(x+3) ≥ 0

ดังนั้น x ≤ -3 หรือ x ≥ 3

จาก x - 2 ≠ 0 ได้ x ≠ 2

คำตอบ: โดเมน = (-∞,-3] ∪ [3,+∞)

ตัวอย่างที่ 2: ฟังก์ชันประกอบ

โจทย์: ให้ f(x) = x + 1 และ g(x) = x²

จงหา (f ∘ g)(2) และ (g ∘ f)(2)

วิธีทำ:

หา (f ∘ g)(2):

(f ∘ g)(x) = f(g(x)) = f(x²) = x² + 1

(f ∘ g)(2) = 2² + 1 = 5

หา (g ∘ f)(2):

(g ∘ f)(x) = g(f(x)) = g(x + 1) = (x + 1)²

(g ∘ f)(2) = (2 + 1)² = 9

ตัวอย่างที่ 3: ฟังก์ชันผกผัน

โจทย์: หาฟังก์ชันผกผันของ f(x) = (2x - 1)/(x + 3)

วิธีทำ:

ให้ y = (2x - 1)/(x + 3)

สลับ x และ y: x = (2y - 1)/(y + 3)

x(y + 3) = 2y - 1

xy + 3x = 2y - 1

xy - 2y = -1 - 3x

y(x - 2) = -1 - 3x

y = (-1 - 3x)/(x - 2)

คำตอบ: f⁻¹(x) = (-3x - 1)/(x - 2)

ตัวอย่างที่ 4: การหาเรนจ์

โจทย์: จงหาเรนจ์ของ f(x) = x² - 4x + 7

วิธีทำ:

เขียนในรูปมาตรฐาน:

f(x) = x² - 4x + 7

= (x² - 4x + 4) + 7 - 4

= (x - 2)² + 3

เนื่องจาก (x - 2)² ≥ 0

ดังนั้น f(x) ≥ 3

คำตอบ: เรนจ์ = [3,+∞)

11. แบบฝึกหัด

แบบทดสอบความเข้าใจ
ข้อ 1: ข้อใดเป็นฟังก์ชัน?
  • ก) x² + y² = 1
  • ข) y = x² + 1
  • ค) x = y²
  • ง) |x| + |y| = 1
ข้อ 2: โดเมนของ f(x) = √(x - 2)/(x - 5) คือข้อใด?
  • ก) x ≥ 2
  • ข) x ≥ 2, x ≠ 5
  • ค) x > 2
  • ง) x ≥ 2, x ≠ 2
ข้อ 3: ถ้า f(x) = 2x + 1 และ g(x) = x - 3 แล้ว (f ∘ g)(x) เท่ากับข้อใด?
  • ก) 2x - 2
  • ข) 2x - 5
  • ค) 2x + 4
  • ง) 2x - 6
ข้อ 4: ฟังก์ชันผกผันของ f(x) = 3x - 2 คือข้อใด?
  • ก) f⁻¹(x) = (x - 2)/3
  • ข) f⁻¹(x) = (x + 2)/3
  • ค) f⁻¹(x) = 3x + 2
  • ง) f⁻¹(x) = (3x + 2)

แบบฝึกหัดเพิ่มเติม

ข้อ 5: หาโดเมนและเรนจ์ของ f(x) = √(4 - x²)

ข้อ 6: ให้ f(x) = x² และ g(x) = x + 1 จงหา (f ∘ g)(x) และ (g ∘ f)(x)

ข้อ 7: ตรวจสอบว่า f(x) = x³ - 1 มีฟังก์ชันผกผันหรือไม่ ถ้ามีจงหา

ข้อ 8: หาฟังก์ชัน f(x) ถ้า f(2x + 1) = 4x² + 4x + 3

ข้อ 9: เขียนกราฟของ f(x) = |x - 2| + 1

ข้อ 10: หาค่า a เพื่อให้ f(x) = ax² + 2x + 1 มีค่าน้อยสุดเท่ากับ 0

เฉลยแบบฝึกหัดเพิ่มเติม

ข้อ 5: โดเมน = [-2,2], เรนจ์ = [0,2]

ข้อ 6: (f ∘ g)(x) = (x+1)², (g ∘ f)(x) = x² + 1

ข้อ 7: มี f⁻¹(x) = ∛(x + 1)

ข้อ 8: f(x) = x² + 2

ข้อ 10: a = 1
