ตรรกศาสตร์ (Mathematical Logic)

บทพื้นฐานสำคัญของคณิตศาสตร์ ม.4 ที่เป็นรากฐานสำหรับการให้เหตุผลและการคิดเชิงตรรกะ

สารบัญเนื้อหา

  • 3. ตัวเชื่อมประพจน์

1. ความรู้เบื้องต้นเกี่ยวกับตรรกศาสตร์

นิยาม

ตรรกศาสตร์ (Mathematical Logic) คือ ระบบวิชาความรู้ที่เกี่ยวข้องกับความคิดและการให้เหตุผล ใช้เป็นเครื่องมือในการเข้าถึงหลักปรัชญาต่างๆ และเป็นพื้นฐานในหลายสาขาวิชา

ความสำคัญของตรรกศาสตร์

ตรรกศาสตร์เป็นเครื่องมือที่สำคัญใน:

  • การให้เหตุผลทางคณิตศาสตร์
  • การพิสูจน์ทฤษฎีบทต่างๆ
  • การเขียนโปรแกรมคอมพิวเตอร์
  • การแก้ไขปัญหาเชิงตรรกะ
  • การคิดวิเคราะห์อย่างเป็นระบบ
แผนภาพแสดงความสัมพันธ์ของตรรกศาสตร์กับสาขาวิชาต่างๆ

2. ประพจน์ (Proposition)

นิยาม

ประพจน์ (Proposition) คือ ข้อความหรือประโยคที่มีค่าความจริง (T) หรือเท็จ (F) อย่างใดอย่างหนึ่งเท่านั้น และแน่นอน

ตัวอย่างประพจน์
  • "2 + 3 = 5" (จริง)
  • "กรุงเทพฯ เป็นเมืองหลวงของไทย" (จริง)
  • "7 > 10" (เท็จ)
  • "หิมะร้อน" (เท็จ)
  • "วันนี้เป็นวันจันทร์" (จริงหรือเท็จ ขึ้นอยู่กับวัน)
ไม่ใช่ประพจน์
  • "x + 2 = 5" (ข้อความเปิด)
  • "คุณชื่ออะไร?" (คำถาม)
  • "ไปกินข้าวกัน!" (คำสั่ง)
  • "ขอให้โชคดี" (คำอวยพร)
  • "คืนนี้สวยจัง" (ความคิดเห็น)

การแทนประพจน์ด้วยตัวอักษร

เราใช้ตัวอักษรพิมพ์เล็ก เช่น p, q, r, s เป็นต้น แทนประพจน์

ตัวอย่าง

กำหนดให้

  • p : "วันนี้ฝนตก"
  • q : "ฉันจะไปโรงเรียน"
  • r : "2 + 2 = 4"
การแยกประเภทข้อความเป็นประพจน์และไม่ใช่ประพจน์

3. ตัวเชื่อมประพจน์

ตัวเชื่อมประพจน์เป็นเครื่องมือที่ใช้เชื่อมประพจน์สองประพจน์หรือมากกว่าให้เป็นประพจน์ใหม่

1. การปฏิเสธ (Negation)

สัญลักษณ์: ∼p หรือ ¬p

อ่านว่า: "ไม่ p" หรือ "not p"

ค่าความจริง: ตรงข้ามกับ p

2. การเชื่อม และ (Conjunction)

สัญลักษณ์: p ∧ q

อ่านว่า: "p และ q"

ค่าความจริง: จริงก็ต่อเมื่อ p และ q จริงทั้งคู่

3. การเชื่อม หรือ (Disjunction)

สัญลักษณ์: p ∨ q

อ่านว่า: "p หรือ q"

ค่าความจริง: จริงก็ต่อเมื่อ p หรือ q หรือทั้งคู่จริง

4. ถ้า...แล้ว (Implication)

สัญลักษณ์: p → q

อ่านว่า: "ถ้า p แล้ว q"

ค่าความจริง: เท็จก็ต่อเมื่อ p จริงและ q เท็จ

5. ถ้าและต่อเมื่อ (Biconditional)

สัญลักษณ์: p ↔ q

อ่านว่า: "p ถ้าและต่อเมื่อ q"

ค่าความจริง: จริงก็ต่อเมื่อ p และ q มีค่าความจริงเหมือนกัน

สรุปตัวเชื่อมประพจน์ทั้ง 5 แบบ พร้อมสัญลักษณ์และความหมาย

4. ตารางค่าความจริง

ตารางค่าความจริง

เป็นตารางที่แสดงค่าความจริงของประพจน์ที่เกิดจากการเชื่อมประพจน์ในทุกกรณีที่เป็นไปได้

ตารางค่าความจริงพื้นฐาน

p q ∼p p ∧ q p ∨ q p → q p ↔ q
T T F T T T T
T F F F T F F
F T T F T T F
F F T F F T T
วิธีการสร้างตารางค่าความจริง
  1. นับจำนวนตัวแปรประพจน์ (n ตัว)
  2. จำนวนแถวจะเป็น 2ⁿ แถว
  3. เขียนค่าความจริงของตัวแปรทั้งหมด
  4. คำนวณค่าความจริงของประพจน์ย่อยก่อน
  5. คำนวณค่าความจริงของประพจน์หลัก
ตัวอย่าง

สร้างตารางค่าความจริงของ (p ∧ q) → ∼r

ตัวแปรประพจน์: p, q, r (3 ตัว)

จำนวนแถว: 2³ = 8 แถว

5. ประพจน์สมมูล

ประพจน์สมมูล

ประพจน์สองประพจน์ P และ Q เรียกว่า สมมูล ถ้าค่าความจริงของ P และ Q เหมือนกันในทุกกรณี เขียนแทนด้วย P ≡ Q

กฎของ De Morgan
  • ∼(p ∧ q) ≡ (∼p ∨ ∼q)
  • ∼(p ∨ q) ≡ (∼p ∧ ∼q)
กฎการแจกแจง (Distributive Laws)
  • p ∧ (q ∨ r) ≡ (p ∧ q) ∨ (p ∧ r)
  • p ∨ (q ∧ r) ≡ (p ∨ q) ∧ (p ∨ r)
กฎการสับเปลี่ยน (Commutative Laws)
  • p ∧ q ≡ q ∧ p
  • p ∨ q ≡ q ∨ p
กฎการเปลี่ยนหมู่ (Associative Laws)
  • (p ∧ q) ∧ r ≡ p ∧ (q ∧ r)
  • (p ∨ q) ∨ r ≡ p ∨ (q ∨ r)
กฎเอกลักษณ์ (Identity Laws)
  • p ∧ T ≡ p
  • p ∨ F ≡ p
  • p ∧ F ≡ F
  • p ∨ T ≡ T
กฎการปฏิเสธคู่ (Double Negation)
  • ∼(∼p) ≡ p
สรุปกฎสำคัญของประพจน์สมมูลในตรรกศาสตร์

การแปลงประพจน์เงื่อนไข

p → q ≡ ∼p ∨ q
p ↔ q ≡ (p → q) ∧ (q → p)
p ↔ q ≡ (p ∧ q) ∨ (∼p ∧ ∼q)

6. สัจนิรันดร์และพยานนิรันดร์

สัจนิรันดร์ (Tautology)

ประพจน์ที่มีค่าความจริงเป็น T ในทุกกรณี

ตัวอย่าง
  • p ∨ ∼p
  • (p → q) ≡ (∼p ∨ q)
  • p → p
พยานนิรันดร์ (Contradiction)

ประพจน์ที่มีค่าความจริงเป็น F ในทุกกรณี

ตัวอย่าง
  • p ∧ ∼p
  • (p ∧ q) ∧ ∼(p ∧ q)
  • p ∧ ∼p ∧ q
ประพจน์เสริมเติม (Contingency)

ประพจน์ที่ไม่ใช่สัจนิรันดร์และไม่ใช่พยานนิรันดร์ (มีค่าความจริงทั้ง T และ F)

การตรวจสอบสัจนิรันดร์และพยานนิรันดร์

วิธีการ
  1. สร้างตารางค่าความจริง
  2. ตรวจสอบคอลัมน์สุดท้าย
  3. ถ้าเป็น T ทั้งหมด → สัจนิรันดร์
  4. ถ้าเป็น F ทั้งหมด → พยานนิรันดร์
  5. ถ้ามีทั้ง T และ F → ประพจน์เสริมเติม

7. การอ้างเหตุผล

การอ้างเหตุผล (Logical Reasoning)

กระบวนการหาผลสรุปจากข้อมูลหรือสมมติฐานที่กำหนดให้ โดยใช้หลักการของตรรกศาสตร์

รูปแบบการอ้างเหตุผลที่สำคัญ

Modus Ponens

รูปแบบ:

ถ้า p → q และ p จริง
แล้ว q จริง
Modus Tollens

รูปแบบ:

ถ้า p → q และ ∼q จริง
แล้ว ∼p จริง
Hypothetical Syllogism

รูปแบบ:

ถ้า p → q และ q → r จริง
แล้ว p → r จริง
Disjunctive Syllogism

รูปแบบ:

ถ้า p ∨ q และ ∼p จริง
แล้ว q จริง
รูปแบบการอ้างเหตุผลที่ใช้บ่อยในตรรกศาสตร์

การตรวจสอบความถูกต้องของการอ้างเหตุผล

วิธีการ
  1. เขียนประพจน์ในรูป (สมมติฐาน₁ ∧ สมมติฐาน₂ ∧ ...) → ผลสรุป
  2. สร้างตารางค่าความจริง
  3. ถ้าประพจน์นี้เป็นสัจนิรันดร์ การอ้างเหตุผลถูกต้อง
  4. ถ้าไม่เป็นสัจนิรันดร์ การอ้างเหตุผลไม่ถูกต้อง

8. ตัวอย่างโจทย์

ตัวอย่างที่ 1: การสร้างตารางค่าความจริง

โจทย์: สร้างตารางค่าความจริงของ (p → q) ∧ (∼q → ∼p)

วิธีทำ:

p q ∼p ∼q p → q ∼q → ∼p (p → q) ∧ (∼q → ∼p)
TTFFTTT
TFFTFFF
FTTFTTT
FFTTTTT

สังเกต: (p → q) ≡ (∼q → ∼p) เป็นประพจน์สมมูล

ตัวอย่างที่ 2: การพิสูจน์ประพจน์สมมูล

โจทย์: พิสูจน์ว่า ∼(p ∧ q) ≡ (∼p ∨ ∼q)

วิธีทำ: ใช้ตารางค่าความจริง

p q p ∧ q ∼(p ∧ q) ∼p ∼q ∼p ∨ ∼q
TTTFFFF
TFFTFTT
FTFTTFT
FFFTTTT

สรุป: คอลัมน์ที่ 4 และ 7 เหมือนกัน ดังนั้น ∼(p ∧ q) ≡ (∼p ∨ ∼q)

ตัวอย่างที่ 3: การตรวจสอบสัจนิรันดร์

โจทย์: ตรวจสอบว่า (p → q) → ((∼q → ∼p)) เป็นสัจนิรันดร์หรือไม่

วิธีทำ:

  1. สร้างตารางค่าความจริง
  2. เนื่องจาก (p → q) ≡ (∼q → ∼p)
  3. ประพจน์จะเป็น (p → q) → (p → q)
  4. ซึ่งมีรูป A → A ที่เป็นสัจนิรันดร์เสมอ

คำตอบ: ใช่ เป็นสัจนิรันดร์

ตัวอย่างที่ 4: การอ้างเหตุผล

โจทย์: ตรวจสอบการอ้างเหตุผลต่อไปนี้

ถ้าฉันเรียนดี ฉันจะได้เกรด A

ฉันไม่ได้เกรด A

ดังนั้น ฉันไม่ได้เรียนดี

วิธีทำ:

กำหนดให้ p: ฉันเรียนดี, q: ฉันได้เกรด A

สมมติฐาน: p → q, ∼q

ผลสรุป: ∼p

รูปแบบ: [(p → q) ∧ ∼q] → ∼p

คำตอบ: นี้คือ Modus Tollens ซึ่งถูกต้อง

รวมโจทย์ตรรกศาสตร์พร้อมเฉลยละเอียด

9. แบบฝึกหัด

แบบทดสอบความเข้าใจ
ข้อ 1: ข้อใดต่อไปนี้เป็นประพจน์?
  • ก) คุณชื่อะไร?
  • ข) 5 + 3 = 8
  • ค) x + 2 > 10
  • ง) ช่วยปิดประตูหน่อย
ข้อ 2: ถ้า p เท็จและ q จริง แล้ว p → q จะมีค่าความจริงเป็นอะไร?
  • ก) จริง
  • ข) เท็จ
  • ค) ไม่สามารถหาได้
  • ง) ขึ้นอยู่กับบริบท
ข้อ 3: ประพจน์ใดต่อไปนี้เป็นสัจนิรันดร์?
  • ก) p ∧ q
  • ข) p ∨ ∼p
  • ค) p → q
  • ง) p ∧ ∼p
ข้อ 4: กฎของ De Morgan ข้อใดถูกต้อง?
  • ก) ∼(p ∧ q) ≡ (∼p ∨ ∼q)
  • ข) ∼(p ∧ q) ≡ (∼p ∧ ∼q)
  • ค) ∼(p ∨ q) ≡ (∼p ∨ ∼q)
  • ง) ∼(p → q) ≡ (∼p → ∼q)
ข้อ 5: การอ้างเหตุผลแบบ Modus Ponens คือ?
  • ก) จาก p → q และ p ได้ q
  • ข) จาก p → q และ ∼q ได้ ∼p
  • ค) จาก p → q และ q ได้ p
  • ง) จาก p → q และ ∼p ได้ ∼q

แบบฝึกหัดเพิ่มเติม

ข้อ 6: สร้างตารางค่าความจริงของ (p ∧ q) → (p ∨ q)

ข้อ 7: พิสูจน์ว่า (p → q) ≡ (∼p ∨ q)

ข้อ 8: ตรวจสอบว่า [(p → q) ∧ (q → r)] → (p → r) เป็นสัจนิรันดร์หรือไม่

ข้อ 9: เขียนการปฏิเสธของ "ถ้าฝนตก แล้วถนนเปียก"

ข้อ 10: ตรวจสอบการอ้างเหตุผล: "ถ้าเขาเรียนดี เขาจะสอบผ่าน เขาสอบผ่าน ดังนั้นเขาเรียนดี"

เฉลยแบบฝึกหัดเพิ่มเติม

ข้อ 6: เป็นสัจนิรันดร์ (เพราะ p ∧ q → p ∨ q เป็นจริงเสมอ)

ข้อ 7: ใช้ตารางค่าความจริงจะได้ผลลัพธ์เหมือนกันทุกกรณี

ข้อ 8: เป็นสัจนิรันดร์ (นี้คือ Hypothetical Syllogism)

ข้อ 9: "ฝนตก และ ถนนไม่เปียก" หรือ p ∧ ∼q

ข้อ 10: ไม่ถูกต้อง (นี้คือ Fallacy of Affirming the Consequent)

แนวข้อสอบ O-NET และ PAT

หัวข้อที่ควรทำความเข้าใจ
  • การสร้างตารางค่าความจริงสำหรับประพจน์ที่ซับซ้อน
  • การใช้กฎของ De Morgan ในการแปลงประพจน์
  • การตรวจสอบสัจนิรันดร์และพยานนิรันดร์
  • การอ้างเหตุผลและการตรวจสอบความถูกต้อง
  • การแปลงประโยคภาษาไทยเป็นสัญลักษณ์ตรรกศาสตร์
เทคนิคและเคล็ดลับในการสอบตรรกศาสตร์

  • การให้เหตุผลในการอภิปราย

  • ธุรกิจ: การวิเคราะห์ความเสี่ยง

เทคนิคการจำและการใช้งาน

เคล็ดลับสำคัญ
  1. การจำตารางค่าความจริง: จำว่า "และ" จริงเมื่อทั้งคู่จริง "หรือ" เท็จเมื่อทั้งคู่เท็จ
  2. ประพจน์เงื่อนไข: p → q เท็จเฉพาะเมื่อ p จริง q เท็จ
  3. กฎ De Morgan: "ไม่ และ" = "ไม่ หรือ ไม่", "ไม่ หรือ" = "ไม่ และ ไม่"
  4. สัจนิรันดร์: มักเป็น p ∨ ∼p หรือ A → A
  5. พยานนิรันดร์: มักเป็น p ∧ ∼p
ข้อผิดพลาดที่พบบ่อย
  • สับสนระหว่าง "หรือ" ในภาษาไทยกับ "หรือ" ทางตรรกศาสตร์
  • คิดว่า p → q เหมือนกับ q → p
  • ใช้ Affirming the Consequent ซึ่งเป็นการอ้างเหตุผลผิด
  • สับสนระหว่างประพจน์สมมูลกับประพจน์เงื่อนไข
  • ลืมใช้วงเล็บในประพจน์ที่ซับซ้อน
