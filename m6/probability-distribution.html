1. ความหมายของตัวแปรสุ่ม

ภาพรวมเนื้อหาตัวแปรสุ่มและการแจกแจงความน่าจะเป็น ม.6
นิยาม: ตัวแปรสุ่ม (Random Variable)

ตัวแปรสุ่ม คือ ฟังก์ชันจากปริภูมิตัวอย่างของการทดลองสุ่มไปยังเซตของจำนวนจริง

มักจะใช้ X, Y, Z (ตัวอักษรพิมพ์ใหญ่) แทนตัวแปรสุ่ม และใช้ x, y, z (ตัวอักษรพิมพ์เล็ก) แทนค่าของตัวแปรสุ่ม

ตัวอย่างตัวแปรสุ่ม

ตัวอย่างที่ 1: การโยนเหรียญ

โยนเหรียญเที่ยงตรง 1 เหรียญ 2 ครั้ง กำหนดให้ X คือ จำนวนครั้งที่เหรียญขึ้นหัว

ปริภูมิตัวอย่าง: S = {TT, HT, TH, HH}

ค่าที่เป็นไปได้ของ X:

  • TT → X = 0 (เหรียญขึ้นหัว 0 ครั้ง)
  • TH, HT → X = 1 (เหรียญขึ้นหัว 1 ครั้ง)
  • HH → X = 2 (เหรียญขึ้นหัว 2 ครั้ง)

ดังนั้น X สามารถมีค่า 0, 1, 2

ตัวแปรสุ่มไม่ต่อเนื่อง (Discrete Random Variable)

ตัวแปรสุ่มที่มีค่าที่เป็นไปได้จำนวนจำกัด หรือนับได้อนันต์

ตัวอย่าง: จำนวนลูก จำนวนครั้งที่ชนะ จำนวนรถที่ผ่าน

ตัวแปรสุ่มต่อเนื่อง (Continuous Random Variable)

ตัวแปรสุ่มที่มีค่าที่เป็นไปได้เป็นช่วงของจำนวนจริง

ตัวอย่าง: น้ำหนัก ส่วนสูง เวลา อุณหภูมิ

2. การแจกแจงความน่าจะเป็นของตัวแปรสุ่มไม่ต่อเนื่อง

การแจกแจงความน่าจะเป็น

การนำความน่าจะเป็นของการเกิดค่าแต่ละค่าที่เป็นไปได้ทั้งหมดของตัวแปรสุ่ม มาแสดงในรูปตารางหรือกราฟ

ตัวอย่างการเขียนกราฟแสดงการแจกแจงความน่าจะเป็น

ตัวอย่างการแจกแจงความน่าจะเป็น

ตัวอย่าง: การโยนเหรียญ 2 ครั้ง

กำหนดให้ X คือ จำนวนครั้งที่เหรียญขึ้นหัว

P(X = x) 1/4 2/4 1/4

1. P(X = xi) ≥ 0 สำหรับทุก i

2. ∑ P(X = xi) = 1

การแจกแจงเอกรูปไม่ต่อเนื่อง

การแจกแจงเอกรูปไม่ต่อเนื่อง
การแจกแจงเอกรูปไม่ต่อเนื่อง

ตัวแปรสุ่ม X มีการแจกแจงเอกรูปไม่ต่อเนื่อง หาก X มีค่าที่เป็นไปได้ n ค่า และแต่ละค่ามีความน่าจะเป็นเท่ากัน

P(X = xi) = 1/n สำหรับทุก i = 1, 2, ..., n

3. ค่าคาดหมายและส่วนเบี่ยงเบนมาตรฐาน

สูตรค่าคาดหมายของตัวแปรสุ่มไม่ต่อเนื่อง
ค่าคาดหมาย (Expected Value)

ค่าคาดหมายของตัวแปรสุ่ม X แทนด้วย E(X) หรือ μX

μX = E(X) = ∑ xi · P(X = xi)
ความแปรปรวน (Variance)

ความแปรปรวนของตัวแปรสุ่ม X แทนด้วย Var(X) หรือ σX²

σX² = ∑ (xi - μX)² · P(X = xi)
ส่วนเบี่ยงเบนมาตรฐานของตัวแปรสุ่มไม่ต่อเนื่อง
ส่วนเบี่ยงเบนมาตรฐาน (Standard Deviation)
σX = √(σX²) = √(∑ (xi - μX)² · P(X = xi))
ตัวอย่างการคำนวณ
ตัวอย่างการคำนวณค่าคาดหมายจากตาราง

4. การแจกแจงทวินาม

การแจกแจงทวินาม (Binomial Distribution)

การแจกแจงความน่าจะเป็นของตัวแปรสุ่ม X ซึ่งเป็นจำนวนครั้งของการเกิดผลสำเร็จจากการทดลองสุ่ม n ครั้ง เป็นอิสระกัน

เขียนแทนด้วย X ~ B(n, p)

สูตรและสมบัติของการแจกแจงทวินาม
เงื่อนไขของการแจกแจงทวินาม
  • เป็นการทำการทดลองซ้ำ ๆ กัน n ครั้ง
  • แต่ละครั้งเป็นอิสระต่อกัน
  • ผลที่ได้มี 2 แบบ: สำเร็จ หรือ ไม่สำเร็จ
  • ความน่าจะเป็นที่สำเร็จ = p (คงที่)
  • ความน่าจะเป็นที่ไม่สำเร็จ = 1 - p
สูตรการแจกแจงทวินาม:

P(X = x) = C(n,x) × px × (1-p)n-x

โดยที่ x = 0, 1, 2, ..., n

ค่าคาดหมาย: μX = np
ความแปรปรวน: σX² = np(1-p)
ส่วนเบี่ยงเบนมาตรฐาน: σX = √(np(1-p))
ตัวอย่าง: การเป่ายิงฉุบ

โจทย์: น้องปั้นเป่ายิงฉุบกับเพื่อน โอกาสที่จะชนะในแต่ละครั้งคือ 0.4 หากเป่ายิงฉุบ 6 ครั้ง จงหา:

วิธีทำ: กำหนด X = จำนวนครั้งที่ชนะ, n = 6, p = 0.4

1. ความน่าจะเป็นที่ชนะอย่างน้อย 1 ครั้ง

P(X ≥ 1) = 1 - P(X = 0) = 1 - C(6,0) × (0.4)⁰ × (0.6)⁶ ≈ 0.95

2. ค่าคาดหมาย

μX = np = 6 × 0.4 = 2.4 ครั้ง

3. ส่วนเบี่ยงเบนมาตรฐาน

σX = √(np(1-p)) = √(6 × 0.4 × 0.6) = 1.2 ครั้ง

5. การแจกแจงความน่าจะเป็นของตัวแปรสุ่มต่อเนื่อง

เส้นโค้งความหนาแน่นของตัวแปรสุ่มต่อเนื่อง
เส้นโค้งความหนาแน่น (Density Curve)

กราฟของการแจกแจงความน่าจะเป็นของตัวแปรสุ่มต่อเนื่องเป็นเส้นโค้ง เรียกว่า เส้นโค้งความหนาแน่น

สมบัติของเส้นโค้งความหนาแน่น
  • f(x) ≥ 0 สำหรับทุก x
  • พื้นที่ใต้เส้นโค้งทั้งหมด = 1
  • P(a < X < b) = พื้นที่ใต้เส้นโค้งระหว่าง a และ b
  • P(X = a) = 0 สำหรับค่า a ใดๆ
การหาความน่าจะเป็น
P(a ≤ X ≤ b) = ∫ab f(x) dx

โดยที่ f(x) คือ ฟังก์ชันความหนาแน่น

ตัวอย่าง: การแจกแจงเอกรูปต่อเนื่อง
การแจกแจงเอกรูปต่อเนื่องบนช่วง [a, b]
การแจกแจงเอกรูปต่อเนื่องบนช่วง [a, b]:

f(x) = 1/(b-a) สำหรับ a ≤ x ≤ b
f(x) = 0 สำหรับ x < a หรือ x > b

ค่าคาดหมาย: μ = (a+b)/2
ความแปรปรวน: σ² = (b-a)²/12

6. การแจกแจงปกติมาตรฐาน

กราฟและสมบัติของการแจกแจงปกติมาตรฐาน
การแจกแจงปกติมาตรฐาน (Standard Normal Distribution)

ตัวแปรสุ่ม Z มีการแจกแจงปกติมาตรฐาน หากมีค่าคาดหมาย μ = 0 และส่วนเบี่ยงเบนมาตรฐาน σ = 1

เขียนแทนด้วย Z ~ N(0, 1)

สมบัติของการแจกแจงปกติมาตรฐาน
  • กราฟเป็นรูประฆังคว่ำ สมมาตรรอบ Z = 0
  • P(Z < 0) = P(Z > 0) = 0.5
  • P(-a < Z < a) = 2Φ(a) - 1
  • P(Z > a) = 1 - Φ(a)
  • P(Z < -a) = Φ(-a) = 1 - Φ(a)
การใช้ตารางการแจกแจงปกติมาตรฐาน

Φ(z) = P(Z ≤ z) โดยที่ Z ~ N(0, 1)

P(a < Z < b) = Φ(b) - Φ(a)
วิธีการใช้ตารางการแจกแจงปกติมาตรฐาน
ตัวอย่างการคำนวณ

ตัวอย่างที่ 1: หา P(-1.5 < Z < 2.3) โดยที่ Z ~ N(0, 1)

วิธีทำ:

P(-1.5 < Z < 2.3) = Φ(2.3) - Φ(-1.5)

= Φ(2.3) - [1 - Φ(1.5)]

= 0.9893 - (1 - 0.9332)

= 0.9893 - 0.0668 = 0.9225

7. การแจกแจงปกติ

การแจกแจงปกติทั่วไปและการมาตรฐานization
การแจกแจงปกติ (Normal Distribution)

ตัวแปรสุ่ม X มีการแจกแจงปกติ หากมีค่าคาดหมาย μ และส่วนเบี่ยงเบนมาตรฐาน σ

เขียนแทนด้วย X ~ N(μ, σ²)

การมาตรฐานization:

หาก X ~ N(μ, σ²) แล้ว Z = (X - μ)/σ ~ N(0, 1)

การหาความน่าจะเป็น:
P(a < X < b) = P((a-μ)/σ < Z < (b-μ)/σ)
= Φ((b-μ)/σ) - Φ((a-μ)/σ)
กฎเกณฑ์ 68-95-99.7 (Rule of Thumb)
  • P(μ - σ < X < μ + σ) ≈ 0.68 (68%)
  • P(μ - 2σ < X < μ + 2σ) ≈ 0.95 (95%)
  • P(μ - 3σ < X < μ + 3σ) ≈ 0.997 (99.7%)
ขั้นตอนการแก้โจทย์
  1. ระบุ μ และ σ
  2. เขียนสิ่งที่โจทย์ต้องการ
  3. แปลงเป็นการแจกแจงปกติมาตรฐาน
  4. ใช้ตารางหาคำตอบ
ตัวอย่าง: คะแนนสอบ
ตัวอย่างการประยุกต์ใช้การแจกแจงปกติ

โจทย์: คะแนนสอบของนักเรียนมีการแจกแจงปกติ โดยมีค่าเฉลี่ย 75 คะแนน และส่วนเบี่ยงเบนมาตรฐาน 10 คะแนน

1. ความน่าจะเป็นที่นักเรียนได้คะแนนระหว่าง 70-85 คะแนน

วิธีทำ: X ~ N(75, 10²)

P(70 < X < 85) = P((70-75)/10 < Z < (85-75)/10)

= P(-0.5 < Z < 1.0)

= Φ(1.0) - Φ(-0.5)

= 0.8413 - 0.3085 = 0.5328

2. คะแนนที่ 90% ของนักเรียนได้คะแนนต่ำกว่า

วิธีทำ: หา k ที่ P(X < k) = 0.90

P(Z < (k-75)/10) = 0.90

จากตาราง Φ(1.28) = 0.90

ดังนั้น (k-75)/10 = 1.28

k = 75 + 12.8 = 87.8 คะแนน

8. ตัวอย่างโจทย์

ตัวอย่างโจทย์และแนวทางการแก้ปัญหา
ตัวอย่างที่ 1: การแจกแจงทวินาม

โจทย์: โยนลูกเต๋า 8 ครั้ง กำหนดให้ X คือ จำนวนครั้งที่ได้แต้ม 6

1. หา P(X = 2)

วิธีทำ: X ~ B(8, 1/6)

P(X = 2) = C(8,2) × (1/6)² × (5/6)⁶

= 28 × (1/36) × (15625/46656)

≈ 0.2605

2. หาค่าคาดหมายและส่วนเบี่ยงเบนมาตรฐาน

μ = np = 8 × (1/6) = 4/3 ≈ 1.33

σ = √(np(1-p)) = √(8 × 1/6 × 5/6) ≈ 1.05

ตัวอย่างที่ 2: การแจกแจงปกติ

โจทย์: น้ำหนักของนักเรียน ม.6 มีการแจกแจงปกติ ค่าเฉลี่ย 60 กก. ส่วนเบี่ยงเบนมาตรฐาน 8 กก.

1. หาความน่าจะเป็นที่นักเรียนหนักเกิน 70 กก.

วิธีทำ: X ~ N(60, 8²)

P(X > 70) = P(Z > (70-60)/8)

= P(Z > 1.25)

= 1 - Φ(1.25) = 1 - 0.8944 = 0.1056

2. หาน้ำหนักที่ 25% ของนักเรียนหนักน้อยกว่า

หา k ที่ P(X < k) = 0.25

P(Z < (k-60)/8) = 0.25

จากตาราง Φ(-0.67) = 0.25

(k-60)/8 = -0.67

k = 60 - 5.36 = 54.64 กก.

ตัวอย่างที่ 3: การแจกแจงเอกรูป

โจทย์: รถเมล์มาถึงป้ายแบบสุ่มในช่วง 12:00-12:15 น. หาความน่าจะเป็นที่รถเมล์จะมาในช่วง 12:05-12:10 น.

วิธีทำ: X ~ Uniform[0, 15]

f(x) = 1/15 สำหรับ 0 ≤ x ≤ 15

P(5 < X < 10) = ∫₅¹⁰ (1/15) dx

= (1/15) × (10 - 5) = 5/15 = 1/3

ตัวอย่างที่ 4: การประมาณทวินามด้วยปกติ

โจทย์: โยนเหรียญ 100 ครั้ง หาความน่าจะเป็นที่ได้หัวมากกว่า 60 ครั้ง

วิธีทำ: X ~ B(100, 0.5)

เนื่องจาก n ใหญ่ จึงประมาณด้วยการแจกแจงปกติ

μ = np = 100 × 0.5 = 50

σ = √(np(1-p)) = √(100 × 0.5 × 0.5) = 5

P(X > 60) ≈ P(Y > 60.5) โดย Y ~ N(50, 5²)

= P(Z > (60.5-50)/5) = P(Z > 2.1)

= 1 - Φ(2.1) = 1 - 0.9821 = 0.0179
