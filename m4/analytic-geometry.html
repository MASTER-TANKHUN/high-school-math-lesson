เรขาคณิตวิเคราะห์และภาคตัดกรวย

การศึกษารูปทรงเรขาคณิตผ่านระบบพิกัดคาร์ทีเซียน และภาคตัดกรวยทั้ง 4 ประเภท

สารบัญเนื้อหา

1. ความรู้เบื้องต้นเกี่ยวกับเรขาคณิตวิเคราะห์

นิยาม

เรขาคณิตวิเคราะห์ (Analytical Geometry) คือ การศึกษาเรขาคณิตโดยใช้พีชคณิตและระบบพิกัด เป็นการเชื่อมโยงระหว่างเรขาคณิตและพีชคณิต

ภาคตัดกรวย (Conic Sections)

ภาคตัดกรวยเกิดจากการตัดพื้นผิวของกรวยวงกลมด้วยระนาบในมุมต่างๆ ได้แก่:

ประเภทของภาคตัดกรวย

วงกลม (Circle)
ตัดขนานกับฐานกรวย
เอลลิปส์ (Ellipse)
ตัดเฉียงไม่ผ่านจุดยอด
พาราโบลา (Parabola)
ตัดขนานกับเส้นกำเนิด
ไฮเพอร์โบลา (Hyperbola)
ตัดผ่านทั้งสองกรวย

2. ระบบพิกัดและสูตรพื้นฐาน

ระบบพิกัดคาร์ทีเซียน

จุด P(x, y) แสดงตำแหน่งในระนาบ

  • x = ระยะจากแกน y (hoแนวนอน)
  • y = ระยะจากแกน x (แนวตั้ง)
  • จุดกำเนิด O(0, 0)
  • แกน x (แนวนอน), แกน y (แนวตั้ง)

สูตรพื้นฐานสำคัญ

ระยะทางระหว่างจุด:
d = √[(x₂-x₁)² + (y₂-y₁)²]

จุดกึ่งกลาง:
M = ((x₁+x₂)/2, (y₁+y₂)/2)

ความชันของเส้นตรง:
m = (y₂-y₁)/(x₂-x₁)

สมการเส้นตรง

รูปมาตรฐาน

Ax + By + C = 0

รูปความชัน-จุด

y - y₁ = m(x - x₁)

รูปความชัน-จุดตัดแกน y

y = mx + c

3. วงกลม (Circle)

นิยาม

วงกลม คือ เซตของจุดทั้งหมดในระนาบที่มีระยะทางเท่ากันจากจุดคงที่หนึ่งจุด (จุดศูนย์กลาง)

สมการวงกลม

รูปมาตรฐาน:
(x - h)² + (y - k)² = r²

รูปทั่วไป:
x² + y² + Dx + Ey + F = 0

โดยที่: ศูนย์กลาง (h, k) และรัศมี r
การแปลงสมการ

จาก x² + y² + Dx + Ey + F = 0

ศูนย์กลาง: (-D/2, -E/2)

รัศมี: r = √[(D² + E² - 4F)/4]

สมบัติของวงกลม

ความสัมพันธ์สำคัญ
  • เส้นผ่านศูนย์กลาง = 2r
  • เส้นรอบวง = 2πr
  • พื้นที่ = πr²
  • เส้นสัมผัสตั้งฉากกับรัศมี
ตัวอย่าง

โจทย์: หาศูนย์กลางและรัศมีของวงกลม

x² + y² - 4x + 6y - 3 = 0

วิธีทำ:

จัดรูปใหม่: (x² - 4x) + (y² + 6y) = 3

ทำให้เป็นกำลังสอง: (x - 2)² + (y + 3)² = 16

คำตอบ: ศูนย์กลาง (2, -3), รัศมี 4

เส้นสัมผัสวงกลม

เส้นสัมผัส ณ จุด (x₁, y₁) บนวงกลม:
(x - h)(x₁ - h) + (y - k)(y₁ - k) = r²

เส้นสัมผัสจากจุดภายนอก:
ความยาวเส้นสัมผัส = √[(x₀ - h)² + (y₀ - k)² - r²]

4. เอลลิปส์ (Ellipse)

นิยาม

เอลลิปส์ คือ เซตของจุดทั้งหมดในระนาบที่ผลรวมของระยะทางจากจุดคงที่สองจุด (โฟกัส) เป็นค่าคงที่

สมการเอลลิปส์

ศูนย์กลางที่จุดกำเนิด:

แกนหลักแนวนอน:
x²/a² + y²/b² = 1 (a > b)

แกนหลักแนวตั้ง:
x²/b² + y²/a² = 1 (a > b)
พารามิเตอร์สำคัญ
  • a = ความยาวกึ่งแกนหลัก
  • b = ความยาวกึ่งแกนรอง
  • c = ระยะจากศูนย์กลางถึงโฟกัส
  • ความสัมพันธ์: c² = a² - b²

สมบัติของเอลลิปส์

จุดสำคัญ
  • โฟกัส: F₁(-c, 0), F₂(c, 0)
  • จุดยอด: (±a, 0), (0, ±b)
  • ความเยื้องศูนย์: e = c/a < 1
  • เส้นไดเรกทริกซ์: x = ±a/e
ตัวอย่าง

โจทย์: หาสมการเอลลิปส์ที่มี

จุดยอดที่ (±5, 0) และ (0, ±3)

วิธีทำ:

a = 5, b = 3 (แกนหลักแนวนอน)

คำตอบ: x²/25 + y²/9 = 1

เอลลิปส์ที่เลื่อนศูนย์กลาง

ศูนย์กลางที่ (h, k):

(x - h)²/a² + (y - k)²/b² = 1

รูปทั่วไป:
Ax² + By² + Dx + Ey + F = 0
(A, B > 0 และ A ≠ B)

5. พาราโบลา (Parabola)

นิยาม

พาราโบลา คือ เซตของจุดทั้งหมดในระนาบที่มีระยะทางจากจุดคงที่หนึ่งจุด (โฟกัส) เท่ากับระยะทางจากเส้นตรงคงที่หนึ่งเส้น (ไดเรกทริกซ์)

สมการพาราโบลา

เปิดขึ้น/ลง (แกนสมมาตรแนวตั้ง):
x² = 4py

เปิดซ้าย/ขวา (แกนสมมาตรแนวนอน):
y² = 4px

โดยที่ p = ระยะจากจุดยอดถึงโฟกัส
ทิศทางการเปิด
  • p > 0: เปิดขึ้น (x²) หรือขวา (y²)
  • p < 0: เปิดลง (x²) หรือซ้าย (y²)
  • โฟกัส: (0, p) หรือ (p, 0)
  • ไดเรกทริกซ์: y = -p หรือ x = -p

สมบัติของพาราโบลา

จุดและเส้นสำคัญ
  • จุดยอด (Vertex): จุดต่ำสุด/สูงสุด
  • แกนสมมาตร: เส้นผ่านจุดยอดและโฟกัส
  • ความเยื้องศูนย์: e = 1
  • เส้นโฟกัล: เส้นผ่านโฟกัสขนานแกน
ตัวอย่าง

โจทย์: หาโฟกัสและไดเรกทริกซ์ของ

x² = 12y

วิธีทำ:

เปรียบเทียบ x² = 4py ได้ 4p = 12

ดังนั้น p = 3

คำตอบ:

โฟกัส: (0, 3)

ไดเรกทริกซ์: y = -3

พาราโบลาที่เลื่อนจุดยอด

จุดยอดที่ (h, k):

เปิดขึ้น/ลง:
(x - h)² = 4p(y - k)

เปิดซ้าย/ขวา:
(y - k)² = 4p(x - h)

รูปทั่วไป: y = ax² + bx + c
ข้อสังเกต

พาราโบลาไม่ใช่ฟังก์ชันเมื่อแกนสมมาตรอยู่ในแนวนอน เพราะ x หนึ่งค่าสามารถมี y ได้หลายค่า

6. ไฮเพอร์โบลา (Hyperbola)

นิยาม

ไฮเพอร์โบลา คือ เซตของจุดทั้งหมดในระนาบที่ค่าสัมบูรณ์ของผลต่างระยะทางจากจุดคงที่สองจุด (โฟกัส) เป็นค่าคงที่

สมการไฮเพอร์โบลา

ศูนย์กลางที่จุดกำเนิด:

แกนหลักแนวนอน:
x²/a² - y²/b² = 1

แกนหลักแนวตั้ง:
y²/a² - x²/b² = 1
พารามิเตอร์สำคัญ
  • a = ความยาวกึ่งแกนหลัก
  • b = ความยาวกึ่งแกนรอง
  • c = ระยะจากศูนย์กลางถึงโฟกัส
  • ความสัมพันธ์: c² = a² + b²

สมบัติของไฮเพอร์โบลา

จุดและเส้นสำคัญ
  • จุดยอด: (±a, 0) หรือ (0, ±a)
  • โฟกัส: (±c, 0) หรือ (0, ±c)
  • ความเยื้องศูนย์: e = c/a > 1
  • เส้นไดเรกทริกซ์: x = ±a/e
ตัวอย่าง

โจทย์: หาจุดยอดและโฟกัสของ

x²/16 - y²/9 = 1

วิธีทำ:

a² = 16 → a = 4

b² = 9 → b = 3

c² = 16 + 9 = 25 → c = 5

คำตอบ:

จุดยอด: (±4, 0)

โฟกัส: (±5, 0)

เส้นกำกับ (Asymptotes)

สำหรับ x²/a² - y²/b² = 1:
y = ±(b/a)x

สำหรับ y²/a² - x²/b² = 1:
y = ±(a/b)x

สำคัญ: ไฮเพอร์โบลาเข้าใกล้แต่ไม่ตัดเส้นกำกับ

รูปแบบไฮเพอร์โบลา

แนวนอน
เปิดซ้าย-ขวา
x²/a² - y²/b² = 1
แนวตั้ง
เปิดขึ้น-ลง
y²/a² - x²/b² = 1

7. การแปลงรูปภาคตัดกรวย

การเลื่อนศูนย์กลาง

วิธีการแปลงสมการ

แทน x ด้วย (x - h) และ y ด้วย (y - k)

โดยที่ (h, k) คือศูนย์กลางใหม่

ตัวอย่างการเลื่อน:

วงกลม:
x² + y² = r² → (x-h)² + (y-k)² = r²

เอลลิปส์:
x²/a² + y²/b² = 1 → (x-h)²/a² + (y-k)²/b² = 1

การหมุนแกน

การแปลงพิกัด

เมื่อหมุนแกนมุม θ:

x = x'cos θ - y'sin θ
y = x'sin θ + y'cos θ
ข้อควรระวัง

การหมุนแกนทำให้เกิดพจน์ xy ในสมการ ซึ่งต้องใช้วิธีการพิเศษในการวิเคราะห์

การจำแนกประเภทภาคตัดกรวย

จากสมการทั่วไป Ax² + Bxy + Cy² + Dx + Ey + F = 0
พิจารณาค่า Δ = B² - 4AC

• Δ < 0: เอลลิปส์ (หรือวงกลม ถ้า A = C และ B = 0)
• Δ = 0: พาราโบลา
• Δ > 0: ไฮเพอร์โบลา

8. ตัวอย่างโจทย์

ตัวอย่างที่ 1: วงกลม

โจทย์: หาสมการวงกลมที่ผ่านจุด A(1, 2), B(3, 4), C(5, 2)

วิธีทำ:

ใช้รูปทั่วไป x² + y² + Dx + Ey + F = 0

แทนค่าจุดทั้งสาม:

  • A: 1 + 4 + D + 2E + F = 0 → D + 2E + F = -5
  • B: 9 + 16 + 3D + 4E + F = 0 → 3D + 4E + F = -25
  • C: 25 + 4 + 5D + 2E + F = 0 → 5D + 2E + F = -29

แก้ระบบสมการได้: D = -6, E = 0, F = 5

คำตอบ: x² + y² - 6x + 5 = 0

ตัวอย่างที่ 2: เอลลิปส์

โจทย์: หาสมการเอลลิปส์ที่มีโฟกัส F₁(-3, 0), F₂(3, 0) และผ่านจุด (0, 4)

วิธีทำ:

c = 3 (ระยะจากศูนย์กลางถึงโฟกัส)

จุด (0, 4) อยู่บนเอลลิปส์:

PF₁ + PF₂ = √(9 + 16) + √(9 + 16) = 2√25 = 10

ดังนั้น 2a = 10 → a = 5

จาก c² = a² - b²: 9 = 25 - b² → b² = 16

คำตอบ: x²/25 + y²/16 = 1

ตัวอย่างที่ 3: พาราโบลา

โจทย์: หาสมการพาราโบลาที่มีจุดยอดที่ (2, -1) และผ่านจุด (4, 3)

วิธีทำ:

ใช้รูป (x - h)² = 4p(y - k)

จุดยอด (h, k) = (2, -1)

แทนจุด (4, 3):

(4 - 2)² = 4p(3 - (-1))

4 = 4p × 4

p = 1/4

คำตอบ: (x - 2)² = (y + 1)

ตัวอย่างที่ 4: ไฮเพอร์โบลา

โจทย์: หาสมการไฮเพอร์โบลาที่มีจุดยอด (±4, 0) และความเยื้องศูนย์ e = 5/4

วิธีทำ:

a = 4 (จากจุดยอด)

e = c/a = 5/4 → c = 5

จาก c² = a² + b²:

25 = 16 + b² → b² = 9

คำตอบ: x²/16 - y²/9 = 1

เส้นกำกับ: y = ±(3/4)x

9. แบบฝึกหัด

แบบทดสอบความเข้าใจ
ข้อ 1: สมการ x² + y² - 4x + 6y - 3 = 0 เป็นวงกลมที่มีศูนย์กลางและรัศมีเท่าไร?
  • ก) ศูนย์กลาง (-2, 3), รัศมี 4
  • ข) ศูนย์กลาง (2, -3), รัศมี 4
  • ค) ศูนย์กลาง (2, -3), รัศมี 16
  • ง) ศูนย์กลาง (-2, 3), รัศมี 2
ข้อ 2: เอลลิปส์ x²/25 + y²/9 = 1 มีโฟกัสอยู่ที่จุดใด?
  • ก) (±4, 0)
  • ข) (0, ±4)
  • ค) (±5, 0)
  • ง) (0, ±3)
ข้อ 3: พาราโบลา y² = 8x มีโฟกัสอยู่ที่จุดใด?
  • ก) (8, 0)
  • ข) (2, 0)
  • ค) (0, 2)
  • ง) (4, 0)
ข้อ 4: ไฮเพอร์โบลา x²/16 - y²/9 = 1 มีเส้นกำกับ (asymptotes) เป็นข้อใด?
  • ก) y = ±(4/3)x
  • ข) y = ±(3/4)x
  • ค) y = ±(9/16)x
  • ง) y = ±(16/9)x
ข้อ 5: สมการ x² - 4y² + 2x + 16y - 19 = 0 เป็นภาคตัดกรวยประเภทใด?
  • ก) วงกลม
  • ข) เอลลิปส์
  • ค) พาราโบลา
  • ง) ไฮเพอร์โบลา

แบบฝึกหัดเพิ่มเติม

ข้อ 6: หาสมการเส้นสัมผัสของวงกลม x² + y² = 25 ณ จุด (3, 4)

ข้อ 7: หาความเยื้องศูนย์ของเอลลิปส์ 4x² + 9y² = 36

ข้อ 8: หาจุดยอดและโฟกัสของพาราโบลา (y - 2)² = -12(x - 3)

ข้อ 9: หาสมการไฮเพอร์โบลาที่มีจุดยอด (0, ±3) และผ่านจุด (2, 5)

ข้อ 10: จำแนกประเภทของ 3x² + 2xy + 3y² - 4x - 4y + 1 = 0

เฉลยแบบฝึกหัดเพิ่มเติม

ข้อ 6: 3x + 4y = 25

ข้อ 7: e = √5/3

ข้อ 8: จุดยอด (3, 2), โฟกัส (0, 2)

ข้อ 9: y²/9 - x²/3 = 1

ข้อ 10: เอลลิปส์ (เพราะ Δ = 4 - 36 = -32 < 0)

สรุปสูตรสำคัญ

ตารางสรุปภาคตัดกรวย

ประเภท สมการมาตรฐาน ความเยื้องศูนย์ โฟกัส
วงกลม x² + y² = r² e = 0 ศูนย์กลาง
เอลลิปส์ x²/a² + y²/b² = 1 0 < e < 1 (±c, 0)
พาราโบลา x² = 4py e = 1 (0, p)
ไฮเพอร์โบลา x²/a² - y²/b² = 1 e > 1 (±c, 0)
