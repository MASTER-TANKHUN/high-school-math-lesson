ลำดับและอนุกรม (Sequences and Series)

บทเรียนสำคัญของคณิตศาสตร์ ม.6 ที่เป็นพื้นฐานสำหรับการศึกษาทางคณิตศาสตร์ขั้นสูงและการประยุกต์ใช้ในชีวิตจริง

สารบัญเนื้อหา

  • 4. ขีดจำกัดของลำดับ (Limit of Sequence)
  • 5. อนุกรมจำกัดและอนุกรมอนันต์

  • 7. การใช้สัญลักษณ์ Sigma
  • 8. การประยุกต์ใช้ในการเงิน

1. ความรู้เบื้องต้นเกี่ยวกับลำดับและอนุกรม

ลำดับ (Sequence)

ลำดับ คือ การเรียงลำดับของจำนวนซึ่งเรียงตามกฎเกณฑ์หรือรูปแบบที่กำหนดไว้ แต่ละจำนวนในลำดับเรียกว่า พจน์ (Term)

อนุกรม (Series)

อนุกรม คือ ผลรวมของพจน์ทั้งหมดในลำดับ สามารถเป็นอนุกรมจำกัด (Finite Series) หรืออนุกรมอนันต์ (Infinite Series)

ความสำคัญของลำดับและอนุกรม

ลำดับและอนุกรมมีความสำคัญใน:

  • การคำนวณทางการเงิน (เงินผ่อน เงินฝาก ดอกเบิ้ยทบต้น)
  • การวิเคราะห์การเติบโตของประชากร
  • การศึกษาแคลคูลัส (Calculus)
  • การประยุกต์ในฟิสิกส์และวิทยาศาสตร์
  • การแก้ปัญหาในชีวิตประจำวัน
แผนภาพแสดงความแตกต่างระหว่างลำดับเลขคณิตและลำดับเรขาคณิต
ตัวอย่างลำดับ
  • ลำดับเลขคณิต: 2, 5, 8, 11, 14, ...
  • ลำดับเรขาคณิต: 3, 6, 12, 24, 48, ...
  • ลำดับฟีโบนัชชี: 1, 1, 2, 3, 5, 8, ...
  • ลำดับกำลังสอง: 1, 4, 9, 16, 25, ...
สัญลักษณ์ที่ใช้
  • aₙ = พจน์ที่ n ของลำดับ
  • a₁, a₂, a₃, ... = ลำดับ
  • Sₙ = ผลรวมของ n พจน์แรก
  • d = ผลต่างร่วม (ลำดับเลขคณิต)
  • r = อัตราส่วนร่วม (ลำดับเรขาคณิต)

2. ลำดับเลขคณิต (Arithmetic Sequence)

นิยาม

ลำดับเลขคณิต คือ ลำดับที่พจน์ใดๆ หลังจากพจน์แรกได้จากการเอาพจน์ก่อนหน้ามาบวกกับค่าคงที่ ค่าคงที่นี้เรียกว่า ผลต่างร่วม (Common Difference)

สูตรลำดับเลขคณิต:
aₙ = a₁ + (n-1)d

โดยที่:
• a₁ = พจน์แรก
• d = ผลต่างร่วม (aₙ₊₁ - aₙ)
• n = ลำดับของพจน์
ตัวอย่างที่ 1

โจทย์: จงหาพจน์ที่ 10 ของลำดับ 3, 7, 11, 15, ...

วิธีทำ:

a₁ = 3, d = 7 - 3 = 4

a₁₀ = a₁ + (10-1)d

a₁₀ = 3 + 9(4) = 3 + 36 = 39

ตอบ: พจน์ที่ 10 คือ 39

ตัวอย่างที่ 2

โจทย์: ลำดับใดที่มี a₁ = 5, d = -2 พจน์ที่ 8 คือเท่าใด?

วิธีทำ:

a₈ = a₁ + (8-1)d

a₈ = 5 + 7(-2)

a₈ = 5 - 14 = -9

ตอบ: พจน์ที่ 8 คือ -9

การหาผลต่างร่วม (d)

ลำดับ ค่า ผลต่างร่วม (d)
2, 5, 8, 11, ...d = 35-2 = 3, 8-5 = 3
10, 7, 4, 1, ...d = -37-10 = -3, 4-7 = -3
1, 1, 1, 1, ...d = 01-1 = 0
สมบัติสำคัญของลำดับเลขคณิต
  • ผลต่างระหว่างพจน์ที่อยู่ติดกันจะเท่ากันเสมอ
  • พจน์กลาง = (พจน์แรก + พจน์สุดท้าย) ÷ 2
  • ถ้า a, b, c เป็นลำดับเลขคณิต แล้ว 2b = a + c
  • กราฟของลำดับเลขคณิตจะเป็นเส้นตรง
สรุปสูตรสำคัญของลำดับและอนุกรม

3. ลำดับเรขาคณิต (Geometric Sequence)

นิยาม

ลำดับเรขาคณิต คือ ลำดับที่พจน์ใดๆ หลังจากพจน์แรกได้จากการเอาพจน์ก่อนหน้ามาคูณกับค่าคงที่ ค่าคงที่นี้เรียกว่า อัตราส่วนร่วม (Common Ratio)

สูตรลำดับเรขาคณิต:
aₙ = a₁ × r^(n-1)

โดยที่:
• a₁ = พจน์แรก
• r = อัตราส่วนร่วม (aₙ₊₁ ÷ aₙ)
• n = ลำดับของพจน์
ตัวอย่างที่ 1

โจทย์: จงหาพจน์ที่ 6 ของลำดับ 2, 6, 18, 54, ...

วิธีทำ:

a₁ = 2, r = 6 ÷ 2 = 3

a₆ = a₁ × r^(6-1)

a₆ = 2 × 3⁵ = 2 × 243 = 486

ตอบ: พจน์ที่ 6 คือ 486

ตัวอย่างที่ 2

โจทย์: ลำดับใดที่มี a₁ = 8, r = 1/2 พจน์ที่ 5 คือเท่าใด?

วิธีทำ:

a₅ = a₁ × r^(5-1)

a₅ = 8 × (1/2)⁴

a₅ = 8 × 1/16 = 1/2

ตอบ: พจน์ที่ 5 คือ 1/2

การหาอัตราส่วนร่วม (r)

ลำดับ อัตราส่วนร่วม (r) การคำนวณ
3, 6, 12, 24, ...r = 26÷3 = 2, 12÷6 = 2
16, 8, 4, 2, ...r = 1/28÷16 = 1/2, 4÷8 = 1/2
2, -6, 18, -54, ...r = -3-6÷2 = -3, 18÷(-6) = -3
สมบัติสำคัญของลำดับเรขาคณิต
  • อัตราส่วนระหว่างพจน์ที่อยู่ติดกันจะเท่ากันเสมอ
  • ถ้า a, b, c เป็นลำดับเรขาคณิต แล้ว b² = ac
  • เมื่อ |r| < 1 ลำดับจะลู่เข้าสู่ 0
  • เมื่อ |r| > 1 ลำดับจะเพิ่มขึ้นอย่างรวดเร็ว
  • เมื่อ r = 1 ลำดับจะเป็นค่าคงที่
  • เมื่อ r = -1 ลำดับจะสลับเครื่องหมาย
เปรียบเทียบลำดับเลขคณิตและลำดับเรขาคณิต

4. ขีดจำกัดของลำดับ (Limit of Sequence)

นิยาม

ขีดจำกัดของลำดับ คือ ค่าที่ลำดับจะเข้าใกล้เมื่อ n เข้าใกล้อนันต์ เขียนแทนด้วย lim(n→∞) aₙ = L โดยที่ L คือค่าขีดจำกัด

กฎสำคัญของขีดจำกัด:
• lim(n→∞) c = c (ค่าคงที่)
• lim(n→∞) 1/n = 0
• lim(n→∞) rⁿ = 0 เมื่อ |r| < 1
• lim(n→∞) (aₙ ± bₙ) = lim aₙ ± lim bₙ
ตัวอย่างที่ 1

โจทย์: หาขีดจำกัดของ aₙ = (3n-2)/n

วิธีทำ:

aₙ = (3n-2)/n = 3 - 2/n

lim(n→∞) aₙ = lim(n→∞) (3 - 2/n)

= 3 - 0 = 3

ตอบ: ขีดจำกัดคือ 3

ตัวอย่างที่ 2

โจทย์: หาขีดจำกัดของ aₙ = (1/2)ⁿ

วิธีทำ:

เนื่องจาก |1/2| < 1

lim(n→∞) (1/2)ⁿ = 0

ตอบ: ขีดจำกัดคือ 0

วิธีการหาขีดจำกัด
  1. แยกตัวประกอบและลดรูป
  2. หารทั้งเศษและส่วนด้วยขั้นสูงสุดของ n
  3. ใช้กฎของขีดจำกัด
  4. พิจารณาค่าของ r ในลำดับเรขาคณิต
ข้อควรระวัง
  • ลำดับที่มีขีดจำกัดเรียกว่า "ลู่เข้า" (Convergent)
  • ลำดับที่ไม่มีขีดจำกัดเรียกว่า "ลู่ออก" (Divergent)
  • ลำดับเลขคณิต (d ≠ 0) จะลู่ออกเสมอ
  • ลำดับเรขาคณิต (|r| ≥ 1, r ≠ 1) จะลู่ออก

5. อนุกรมจำกัดและอนุกรมอนันต์

อนุกรมจำกัด (Finite Series)

คือ ผลรวมของจำนวนพจน์จำกัดในลำดับ เขียนแทนด้วย S₍ = a₁ + a₂ + a₃ + ... + aₙ

อนุกรมอนันต์ (Infinite Series)

คือ ผลรวมของพจน์ทั้งหมดในลำดับอนันต์ เขียนแทนด้วย S = a₁ + a₂ + a₃ + ... หากมีขีดจำกัดจะเรียกว่า "ลู่เข้า" (Convergent)

อนุกรมเลขคณิต (จำกัด)
Sₙ = n/2 [2a₁ + (n-1)d]
หรือ
Sₙ = n/2 (a₁ + aₙ)
อนุกรมเรขาคณิต (จำกัด)
Sₙ = a₁(1-rⁿ)/(1-r)
เมื่อ r ≠ 1
อนุกรมเรขาคณิตอนันต์
S∞ = a₁/(1-r)
เมื่อ |r| < 1

หาก |r| ≥ 1 อนุกรมจะลู่ออก (ไม่มีผลรวม)

ตัวอย่างอนุกรมเลขคณิต

โจทย์: หาผลรวม 2+5+8+11+...+32

วิธีทำ:

a₁ = 2, d = 3, aₙ = 32

หา n: 32 = 2 + (n-1)3

30 = 3(n-1), n = 11

S₁₁ = 11/2 (2+32) = 11×17 = 187

ตัวอย่างอนุกรมเรขาคณิต

โจทย์: หาผลรวม 1+1/2+1/4+1/8+...

วิธีทำ:

a₁ = 1, r = 1/2

เนื่องจาก |1/2| < 1

S∞ = 1/(1-1/2) = 1/(1/2) = 2

สรุปแนวคิดและสูตรของลำดับและอนุกรม

6. อนุกรมเลขคณิตและอนุกรมเรขาคณิต

อนุกรมเลขคณิต

ขั้นตอนการแก้โจทย์
  1. หาพจน์แรก (a₁) และผลต่างร่วม (d)
  2. หาจำนวนพจน์ (n) หากไม่ทราบ
  3. แทนค่าในสูตร Sₙ = n/2[2a₁+(n-1)d]
  4. คำนวณผลรวม

อนุกรมเรขาคณิต

ขั้นตอนการแก้โจทย์
  1. หาพจน์แรก (a₁) และอัตราส่วนร่วม (r)
  2. ตรวจสอบว่า |r| < 1 หรือไม่
  3. เลือกสูตรที่เหมาะสม (จำกัดหรืออนันต์)
  4. คำนวณผลรวม
ตัวอย่างเปรียบเทียบ

อนุกรมเลขคณิต: 3 + 7 + 11 + 15 + ... + 43

a₁ = 3, d = 4, aₙ = 43

43 = 3 + (n-1)4 → n = 11

S₁₁ = 11/2 (3+43) = 253

อนุกรมเรขาคณิต: 3 + 6 + 12 + 24 + ... (5 พจน์)

a₁ = 3, r = 2, n = 5

S₅ = 3(1-2⁵)/(1-2) = 3(-31)/(-1) = 93

ข้อผิดพลาดที่พบบ่อย
  • สับสนระหว่างลำดับและอนุกรม
  • ใช้สูตรผิดประเภท (เลขคณิต/เรขาคณิต)
  • ลืมตรวจสอบเงื่อนไข |r| < 1 สำหรับอนุกรมอนันต์
  • คำนวณจำนวนพจน์ (n) ผิด

7. การใช้สัญลักษณ์ Sigma (∑)

สัญลักษณ์ Sigma

สัญลักษณ์ ∑ (sigma) ใช้แทนการบวก (summation) โดย ∑ₖ₌₁ⁿ aₖ หมายถึง a₁ + a₂ + a₃ + ... + aₙ

สูตรสำคัญ:
• ∑ₖ₌₁ⁿ c = nc (c คือค่าคงที่)
• ∑ₖ₌₁ⁿ k = n(n+1)/2
• ∑ₖ₌₁ⁿ k² = n(n+1)(2n+1)/6
• ∑ₖ₌₁ⁿ k³ = [n(n+1)/2]²
ตัวอย่างที่ 1

โจทย์: คำนวณ ∑ₖ₌₁⁵ (2k+1)

วิธีทำ:

∑ₖ₌₁⁵ (2k+1) = ∑ₖ₌₁⁵ 2k + ∑ₖ₌₁⁵ 1

= 2∑ₖ₌₁⁵ k + 5

= 2 × 5(6)/2 + 5

= 30 + 5 = 35

ตัวอย่างที่ 2

โจทย์: คำนวณ ∑ₖ₌₁⁴ k²

วิธีทำ:

∑ₖ₌₁⁴ k² = 4(4+1)(2×4+1)/6

= 4 × 5 × 9/6

= 180/6 = 30

ตรวจสอบ: 1² + 2² + 3² + 4² = 1 + 4 + 9 + 16 = 30 ✓

สมบัติของ Sigma
  • ∑(aₖ + bₖ) = ∑aₖ + ∑bₖ
  • ∑(c × aₖ) = c × ∑aₖ
  • ∑ₖ₌ₐᵇ aₖ = ∑ₖ₌₁ᵇ aₖ - ∑ₖ₌₁ᵃ⁻¹ aₖ
  • ∑ₖ₌₁ⁿ aₖ = ∑ⱼ₌₁ⁿ aⱼ (เปลี่ยนตัวแปร)

การเขียน Sigma จากลำดับ

ลำดับ รูปแบบ Sigma คำอธิบาย
1, 4, 9, 16, 25∑ₖ₌₁⁵ k²กำลังสองของจำนวนธรรมชาติ
2, 4, 6, 8, 10∑ₖ₌₁⁵ 2kจำนวนคู่
1, 3, 5, 7, 9∑ₖ₌₁⁵ (2k-1)จำนวนคี่
แนวคิดและการประยุกต์ใช้ลำดับเรขาคณิต

8. การประยุกต์ใช้ในการเงิน

1. เงินฝากและดอกเบิ้ย

ดอกเบิ้ยธรรมดา (Simple Interest)

ยอดเงินรวม = เงินต้น + (เงินต้น × อัตราดอกเบิ้ย × เวลา)

A = P(1 + rt)

ดอกเบิ้ยทบต้น (Compound Interest)

ยอดเงินรวม = เงินต้น × (1 + อัตราดอกเบิ้ย)^เวลา

A = P(1 + r)^t

ตัวอย่างเงินฝาก

โจทย์: ฝากเงิน 10,000 บาท อัตราดอกเบิ้ย 5% ต่อปี ทบต้นทุกปี เป็นเวลา 5 ปี

วิธีทำ:

P = 10,000, r = 0.05, t = 5

A = 10,000(1.05)⁵

A = 10,000 × 1.276 = 12,760 บาท

ตัวอย่างเงินผ่อน

โจทย์: ผ่อนเงิน 2,000 บาท/เดือน เป็นเวลา 2 ปี ดอกเบิ้ย 1%/เดือน

วิธีทำ:

R = 2,000, r = 0.01, n = 24

ใช้สูตรอนุกรมเรขาคณิต:

FV = R × [(1+r)ⁿ-1]/r

FV = 2,000 × [(1.01)²⁴-1]/0.01

≈ 54,400 บาท

2. การคำนวณเงินผ่อนซื้อ

สูตรเงินผ่อนคงที่:
PMT = PV × [r(1+r)ⁿ]/[(1+r)ⁿ-1]

โดยที่:
• PMT = ค่าผ่อนต่องวด
• PV = มูลค่าปัจจุบัน
• r = อัตราดอกเบิ้ยต่องวด
• n = จำนวนงวด
ตัวอย่างเงินกู้

โจทย์: กู้เงิน 500,000 บาท ดอกเบิ้ย 6% ต่อปี ผ่อน 5 ปี ค่าผ่อนต่อเดือนเท่าไร?

วิธีทำ:

PV = 500,000, r = 0.06/12 = 0.005, n = 5×12 = 60

PMT = 500,000 × [0.005(1.005)⁶⁰]/[(1.005)⁶⁰-1]

PMT ≈ 9,661 บาท/เดือน

3. การออมเงิน

การออมเงินสม่ำเสมอ

หากออมเงิน R บาทต่อเดือน ในบัญชีที่ให้ดอกเบิ้ย r ต่อเดือน เป็นเวลา n เดือน

FV = R × [(1+r)ⁿ-1]/r
ตัวอย่างการออม

โจทย์: ออมเงิน 3,000 บาท/เดือน ได้ดอกเบิ้ย 3% ต่อปี หลัง 10 ปี จะมีเงินเท่าไร?

วิธีทำ:

R = 3,000, r = 0.03/12 = 0.0025, n = 120

FV = 3,000 × [(1.0025)¹²⁰-1]/0.0025

FV ≈ 419,000 บาท

ข้อควรระวังในการคำนวณการเงิน
  • แปลงอัตราดอกเบิ้ยให้สอดคล้องกับรอบการคำนวณ
  • ระวังการปัดทศนิยมในการคำนวณหลายขั้นตอน
  • เข้าใจความแตกต่างระหว่างดอกเบิ้ยธรรมดาและทบต้น
  • ตรวจสอบหน่วยของเวลา (ปี/เดือน/วัน)

9. ตัวอย่างโจทย์

โจทย์ที่ 1: ลำดับเลขคณิต

โจทย์: ในลำดับเลขคณิต ถ้า a₃ = 11 และ a₇ = 23 จงหา a₁ และ d

วิธีทำ:

a₃ = a₁ + 2d = 11 ... (1)

a₇ = a₁ + 6d = 23 ... (2)

จาก (2) - (1): 4d = 12

∴ d = 3

แทนใน (1): a₁ + 6 = 11

∴ a₁ = 5

ตอบ: a₁ = 5, d = 3

โจทย์ที่ 2: ลำดับเรขาคณิต

โจทย์: ในลำดับเรขาคณิต ถ้า a₂ = 6 และ a₅ = 162 จงหา a₁ และ r

วิธีทำ:

a₂ = a₁r = 6 ... (1)

a₅ = a₁r⁴ = 162 ... (2)

หาร (2) ÷ (1): r³ = 27

∴ r = 3

แทนใน (1): a₁ × 3 = 6

∴ a₁ = 2

ตอบ: a₁ = 2, r = 3

โจทย์ที่ 3: อนุกรมอนันต์

โจทย์: หาผลรวมของ 2 + 2/3 + 2/9 + 2/27 + ...

วิธีทำ:

a₁ = 2, r = 1/3

เนื่องจาก |1/3| < 1

S∞ = a₁/(1-r)

S∞ = 2/(1-1/3) = 2/(2/3) = 3

ตอบ: ผลรวมคือ 3

โจทย์ที่ 4: Sigma

โจทย์: คำนวณ ∑ₖ₌₁¹⁰ (3k² - 2k + 1)

วิธีทำ:

= 3∑ₖ₌₁¹⁰ k² - 2∑ₖ₌₁¹⁰ k + ∑ₖ₌₁¹⁰ 1

= 3 × 10(11)(21)/6 - 2 × 10(11)/2 + 10

= 3 × 385 - 2 × 55 + 10

= 1155 - 110 + 10 = 1055

ตอบ: 1055

โจทย์ที่ 5: การประยุกต์

โจทย์: แบคทีเรียแบ่งตัวเป็น 2 เท่าทุก 20 นาที เริ่มต้น 100 ตัว หลัง 2 ชั่วโมง มีกี่ตัว?

วิธีทำ:

2 ชั่วโมง = 120 นาที

จำนวนรอบการแบ่งตัว = 120 ÷ 20 = 6 รอบ

จำนวนแบคทีเรีย = 100 × 2⁶

= 100 × 64 = 6,400 ตัว

ตอบ: 6,400 ตัว

โจทย์ที่ 6: ขีดจำกัด

โจทย์: หาขีดจำกัดของ aₙ = (5n² - 3n + 1)/(2n² + n - 4)

วิธีทำ:

หารทั้งเศษและส่วนด้วย n²

aₙ = (5 - 3/n + 1/n²)/(2 + 1/n - 4/n²)

lim(n→∞) aₙ = (5 - 0 + 0)/(2 + 0 - 0) = 5/2

ตอบ: 5/2
